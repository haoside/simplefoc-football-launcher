# 短导向筒 A/B 测试矩阵 v1

## 目标
评估短导向筒对以下指标的影响：
- 出球方向一致性
- 直射偏航
- 轻弧线稳定性
- 标准弧线稳定性
- 是否削弱旋转

## 测试变量
### 导向筒
- G0：无导向
- G1：150mm / Ø232
- G2：200mm / Ø232
- G3：250mm / Ø232

### 发射模式
- P1：`preset straight_pass`
- P2：`preset light_left_curve`
- P3：`preset light_right_curve`
- P4：`preset standard_left_curve`
- P5：`preset standard_right_curve`

## 判定原则
### 导向筒保留
满足至少两条：
1. 直射偏航更小
2. 落点离散度更小
3. 轻弧线可重复性更高
4. 标准弧线没有明显被削弱

### 导向筒淘汰
任一明显出现：
1. 球速明显下降
2. 旋转效果被明显削弱
3. 球面摩擦痕迹明显
4. 偏航反而更大
