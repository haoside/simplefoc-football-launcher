8. 标准 5 号足球 ×3–5

2. 130mm 摩擦轮 ×3
3. 联轴器
4. 轴承 / 轴承座

这些需要实测数据后再买：

1. 高电流 SimpleFOC 驱动

4. 现场移动底盘正式件

## 下单后检查

- 拍照记录型号和铭牌。
- 保存购买链接。
- 到货后先录入 `docs/P0A_TEST_LOG_TEMPLATE_V1.md`。
- 电机到货先确认 Hall 线序和轴径。
- 轮子到货先确认硬度、轮孔、跳动。
