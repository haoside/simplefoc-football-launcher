# P0-A 采购 Checklist v1

## 采购目标

只采购能支撑 P0-A 台架 bring-up 的物料：单电机闭环、三电机同步、急停保护、低速单球发射。

不采购 P0-B 高功率件，避免在电源保护和实测数据未闭环前浪费预算。

## 暂缓清单

| 类别 | 原因 |
|---|---|
| 高电流 SimpleFOC 驱动 | 等 P0-A 电流实测和电源保护复核 |
| 24V LiFePO4 40–60Ah | 等 BMS/保险/线径定版 |
| 自动捡球机构 | P0 只预留接口 |
| 多球仓/管道/闸门 | 明确不进 P0 |
| 4000–5000rpm 高速轮组 | 等 110mm/3000rpm 安全验证后再升级 |

## 下单前检查

- 电机是否带 Hall。
- 电机轴径是否匹配联轴器。
- 摩擦轮是否可更换 110/130mm。
- B-G431B-ESC1 只用于低功率 bring-up，不按最终功率采购。
- 急停是否为物理硬切，不是软件按钮。
