# P0-A 采购预研清单 v2

## 目标

先完成低风险台架 bring-up：单电机闭环、三电机同步、低速单球发射、安全链路验证。

P0-A 不追求最终实战功率，采购必须避免把 B-G431B-ESC1 当最终实球高功率驱动。

## A. 必买：控制 / bring-up

| 类别 | 建议 | 数量 | 备注 |
|---|---|---:|---|
| Host 控制 | ESP32-S3 DevKit | 1 | 参数下发、CAN、日志 |
| 电机节点 | B-G431B-ESC1 | 3 | P0-A 低功率闭环/代码验证 |
| CAN | SN65HVD230 / 等效 CAN 模块 | 4 | Host + 3 node |
| 急停 | 自锁蘑菇头急停 | 1–2 | 必须硬切 EN / 主动力链路 |
| 调试 | USB-UART / ST-Link | 若干 | 固件下载与日志 |

## B. 必买：电机 / 反馈

| 类别 | 建议 | 数量 | 备注 |
|---|---|---:|---|
| 电机 | 6374 BLDC 170–190KV 带 Hall | 3 | 8mm 轴优先 |
| Hall 线束 | 与电机匹配 | 3 | P0 必选 |
| 编码器预留 | AS5048A/SPI | 0–3 | 非 P0 必买，可后补 |

## C. 必买：低速台架供电

| 类别 | 建议 | 数量 | 备注 |
|---|---|---:|---|
| 台架电源 | 24V 可调限流电源 | 1 | P0-A 首选，便于限流 |
| 保险 | DC 保险 / 保险座 | 若干 | 总线 + 分路 |
| 线材 | 10AWG / 12AWG / 16–24AWG | 若干 | 动力/低压/信号分开 |
| 接插件 | XT90/XT60/端子排 | 若干 | 按电流等级选 |

## D. 机械 / 安全

| 类别 | 建议 | 数量 | 备注 |
|---|---|---:|---|
| 摩擦轮 | 110mm 高摩擦包胶轮 | 3–6 | P0-A 起步；建议多买备件 |
| 备选轮 | 130mm 高摩擦包胶轮 | 3 | P0-B 候选 |
| 联轴器 | 8mm 轴适配 | 3–6 | 需按电机/轮轴确认 |
| 支撑件 | 轴承座/轴承/铝型材 | 若干 | 轮组调节结构 |
| 防护罩 | 6–8mm PC 板 | 1 套 | 实球前必须装 |
| 足球 | 标准 5 号球 | 3–5 | 不同新旧状态用于测试 |

## E. P0-B 暂缓采购

| 类别 | 暂缓原因 |
|---|---|
| 高电流 SimpleFOC 驱动 | 需要先完成电源保护 v1 复核 |
| 24V LiFePO4 40–60Ah | 需要确认 BMS/保险/线径 |
| 自动捡球机构 | P0 只预留接口 |
| 多球仓/管道/闸门 | 明确不进 P0 |

## 采购策略

1. 先买 P0-A 台架件。
2. 等单电机闭环、电源保护 v1、P0-SIM 参数回填后，再买 P0-B 高功率件。
3. 机械件按 110/130mm 可替换轮组设计，避免返工。
