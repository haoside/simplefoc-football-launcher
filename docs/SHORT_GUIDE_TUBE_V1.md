# 短导向筒尺寸草案 v1

## 设计边界
- 允许：短导向结构，用于上球定位、初始方向稳定
- 不允许：供球管、长管膛、多球导槽
- 不允许：明显摩擦球面，破坏旋转
- 要求：可拆卸，便于 A/B 测试

## 建议尺寸
- 长度：150 ~ 250mm
- 首版建议：200mm
- 内径：230 ~ 235mm
- 首版建议：232mm
- 与球总间隙：约 5 ~ 7.5mm

## 结构建议
- 入口：喇叭口过渡
- 中段：短直筒
- 出口：圆角收边
- 材料：UHMWPE / HDPE / 尼龙 / 光滑 PC 内衬

## A/B 测试组合
1. 无导向
2. 150mm
3. 200mm
4. 250mm

## 验证项
- 出球方向一致性
- 直射偏航
- 轻弧线 / 标准弧线稳定性
- 是否削弱旋转
