# 电机布局几何约束 v1

> 来源：Owner 提供的参考图 + 口述约束
> 最后更新：2026-07-10

## 核心约束

**三电机轴延长线构成等边三角形。**

```
        Motor A (12:00)
           │
           │ shaft axis
           │
     ┌─────┼─────┐
     │  ╲  │  ╱  │  ← 等边三角形边
     │    ╲│╱    │
     │    (ball) │  ← 足球位于三角形内切位置
     │    ╱│╲    │
     │  ╱  │  ╲  │
     └─────┼─────┘
           │
  Motor B ─┘ └─ Motor C
 (8:00)        (4:00)
```

## 几何关系

### 等边三角形参数

| 参数 | 符号 | 说明 |
|------|------|------|
| 三角形边长 | `s` | 电机轴延长线交点之间的距离 |
| 三角形内切圆半径 | `r_in` | `r_in = s × √3 / 6` |
| 三角形外接圆半径 | `r_out` | `r_out = s × √3 / 3` = `2 × r_in` |

### 与物理部件的关系

| 物理量 | 公式 | 说明 |
|--------|------|------|
| 球心到电机轴线垂直距离 | `d = r_in` | 球在三角形内切位置 |
| 电机轴心到球心距离 | `R = r_in + R_wheel + R_ball` | 含摩擦轮半径和球半径 |
| 摩擦轮接触点到球心距离 | `R_ball` | 球半径 |
| 预压量 | `preload = R_wheel - (R - R_ball - r_in)` | 摩擦轮压入球面的深度 |

### 已知参数

| 参数 | 值 | 来源 |
|------|-----|------|
| 球直径 | 220mm (标准5号球) | FIFA标准 |
| 球半径 `R_ball` | 110mm | |
| 摩擦轮直径 | 待定 (建议 90~130mm) | 结构设计后细化 |
| 摩擦轮半径 `R_wheel` | 待定 | |
| 电机型号 | 6374 (φ63mm × 74mm) | P0确认 |

## 约束传递链

```
等边三角形约束
  → 电机角位置: 120° 均布 (A=0°, B=120°, C=240°)
  → 电机轴线方向: 精确指向球心
  → 电机径向距离: r_in + R_wheel + R_ball
  → 三轮接触点对称: 合力方向自然指向球心
  → RPM mixer 120° 公式成立的前提条件
```

## 对固件的影响

**RPM mixer (`can_host_loop.cpp`) 的 120° 差速公式假设三轮对称布局。**

如果电机不满足等边三角形约束：
- 三轮对球的压力不均 → 发射偏转
- 差速计算需要引入非对称补偿项
- 当前固件的 `compute_targets()` 和 `mix120deg()` 不适用

**结论：等边三角形布局是固件控制逻辑的几何前提。结构设计必须保证。**

## CAD 验证检查项

| # | 检查项 | 通过标准 |
|---|--------|---------|
| 1 | 三电机轴延长线是否构成等边三角形 | 三角偏差 < 1mm |
| 2 | 电机轴线是否指向球心 | 角度偏差 < 1° |
| 3 | 三轮接触点是否对称 | 间距偏差 < 2mm |
| 4 | 预压量是否均匀 | 三轮 preload 差 < 1mm |
| 5 | 球通道内径是否匹配 | 220mm ± 2mm |
