# 电机布局几何约束 v1

> 来源：Owner 提供的参考图 + 口述约束
> 最后更新：2026-07-10

## 核心约束

**三电机轴延长线构成等边三角形。电机轴线切向偏置（非径向指向球心）。**

```
        Motor A (12:00)
           │
           │ shaft axis (偏置，不指向圆心)
           │
     ┌─────┼─────┐
     │  ╲  │  ╱  │  ← 等边三角形边 = 轴延长线
     │    ╲│╱    │
     │  (tube)   │  ← 发射筒（中心圆），非球的内切位置
     │    ╱│╲    │
     │  ╱  │  ╲  │
     └─────┼─────┘
           │
  Motor B ─┘ └─ Motor C
 (8:00)        (4:00)
```

## 几何关系

### 等边三角形参数

| 参数 | 符号 | 说明 |
|------|------|------|
| 三角形边长 | `s` | 电机轴延长线交点之间的距离 |
| 三角形内切圆半径 | `r_in` | `r_in = s × √3 / 6` |
| 三角形外接圆半径 | `r_out` | `r_out = s × √3 / 3` = `2 × r_in` |

### 关键几何特征

| 特征 | 说明 |
|------|------|
| 电机轴延长线 | 构成等边三角形的三条边 |
| 轴线方向 | 切向偏置，**不指向发射筒中心** |
| 发射筒位置 | 位于等边三角形中心 |
| 摩擦轮接触 | 轴延长线与筒外壁的切点附近 |
| 力的分解 | 法向力（压紧球）+ 切向力（驱动球旋转/前进） |

## 电机端面设计

### 固定端面（电机本体端）
- **4 螺丝孔固定板**
- 匹配 6374 电机的 4 个安装孔
- 用于将电机固定到机架/支架
- 螺丝规格待定（M3/M4，取决于 6374 法兰孔径）

### 旋转端面（轴/摩擦轮端）
- **非接触保护挡板**
- 不接触旋转部件，留安全间隙
- 防止异物/手指进入旋转区域
- 材料/结构待定（3D打印透明件？钣金？）

## 发射筒结构

**发射筒 = 中心圆柱体，壁面开孔嵌入电机外转子。**

- 发射筒圆柱面开 3 个圆孔（120° 均布）
- 孔径匹配 6374 外转子直径（~63mm）
- 电机外转子圆筒嵌入孔中
- 摩擦轮从外转子伸入筒内接触球
- 发射筒同时承担三重角色：
  1. **球通道** — 标准5号球通过
  2. **电机固定基座** — 外转子嵌入孔中定位
  3. **结构骨架** — 连接三个电机，传递载荷

### 偏置角度的意义

电机轴线不指向球心，而是切向偏置。这意味着：
- 摩擦轮对球施加的力有**切向分量**
- 切向分量 = 球获得前进速度和旋转的来源
- 三轮对称偏置 → 合力方向仍沿发射方向
- 差速控制 → 切向力不等 → 球获得侧旋（弧线球）

## 约束传递链

```
等边三角形约束（轴延长线构成三角形）
  → 电机角位置: 120° 均布 (A=0°, B=120°, C=240°)
  → 轴线切向偏置（不指向球心）
  → 摩擦轮接触力 = 法向（压紧）+ 切向（驱动）
  → 三轮对称偏置 → 合力沿发射方向
  → 差速控制 → 切向力不等 → 侧旋（弧线球）
  → RPM mixer 120° 公式成立的前提条件
```

## 对固件的影响

**RPM mixer (`can_host_loop.cpp`) 的 120° 差速公式假设三轮对称切向偏置布局。**

如果电机不满足等边三角形约束或偏置角度不一致：
- 三轮对球的切向力不均 → 发射偏转
- 差速计算需要引入非对称补偿项
- 当前固件的 `compute_targets()` 和 `mix120deg()` 不适用

**结论：等边三角形布局 + 切向偏置是固件控制逻辑的几何前提。结构设计必须保证。**

## CAD 验证检查项

| # | 检查项 | 通过标准 |
|---|--------|---------|
| 1 | 三电机轴延长线是否构成等边三角形 | 三角偏差 < 1mm |
| 2 | 轴线偏置方向是否一致（切向，非径向） | 角度偏差 < 1° |
| 3 | 三轮接触点是否对称 | 间距偏差 < 2mm |
| 4 | 预压量是否均匀 | 三轮 preload 差 < 1mm |
| 5 | 发射筒内径是否匹配 | 220mm ± 2mm |
| 6 | 偏置角度是否三轮一致 | 角度差 < 1° |
