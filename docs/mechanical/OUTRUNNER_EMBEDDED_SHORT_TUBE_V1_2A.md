# OUTRUNNER_EMBEDDED_SHORT_TUBE_V1_2A

## 新基准收紧
按新图和产品经理提炼，v1.2a 把主线再收紧为：

- **中心是短圆管 / 短环**，不是长发射筒主体
- **3 个外转子围绕圆管环抱**，贴近管壁接触球
- **电机嵌入式 / 夹持式安装**，不是外侧外挂
- **发射方向仍以圆管出口轴线为准**

对应 CAD：
- `cad/assembly/p0-single-ball-launcher-v3/outrunner_embedded_short_tube_v1_2a.scad`

---

## 与 v1.2 的区别

### 1. 管体缩短
v1.2 仍偏“短发射圆管 + 嵌入电机”，但长度还像一个小筒。  
v1.2a 明确改为 **短圆管 / 短环基准**：

```scad
short_tube_len = 76;
```

这更接近图里的结构：
- 中心只有短导向段
- 电机主要围绕短环布置
- 出口方向由短圆管轴线定义

### 2. 电机不外挂
v1.2a 的电机站位统一为：
- 管壁嵌入口袋
- 背部 saddle
- 侧向夹持
- 单侧端面夹板

视觉和结构上都不再是“外挂电机模块”。

### 3. 发射方向明确
CAD 中用绿色箭头表达：
- `+Z` 为圆管出口方向
- 发射方向以圆管出口为准
- 三电机只负责加速/旋转，不改变出口轴线定义

## 结构说明

### 中心短圆管
- 内径仍按 `232mm`
- 作用是定义球通过的中心通道与出口轴线
- 不是长筒导向，不承担长距离导向功能

### 三外转子嵌入
- 三个 6354 外转子 120° 环抱
- 外转子工作面进入球道 `2–3mm`
- 接触逻辑仍然是直接外转子 / 包胶接触球

### 嵌入式夹持安装
每个电机由三部分约束：
1. 管壁嵌入口袋
2. 背部 saddle 支撑
3. 单侧端面夹板

后续实测端面孔距后，只改端夹孔位参数，不改整体排布。

### 单端 45×45 固定
继续保留 owner 指定接口：
- 一端固定
- 4 螺丝
- 45×45 方阵
- 默认 M5 间隙孔

## 工程风险

1. **短圆管导向长度不足**
   - 优点：结构紧凑，符合图
   - 风险：球进入/离开时稳定性依赖三电机同步和入口姿态

2. **嵌入式电机散热更难**
   - 电机贴近管壁
   - 防护罩不能封死
   - 需要预留导风/散热开口

3. **夹持座与真实 6354 外形可能干涉**
   - 端面凸台
   - 线束出口
   - 螺钉头高度
   - 包胶外径

4. **50m 与 180° 弧线仍是仿真目标**
   - 结构已为仿真收紧
   - 但性能必须由转速、摩擦、压紧量、差速控制验证

## 下一步给工程

@全栈工程师 请以 v1.2a 为新主线基准，不再沿 v1.2 长筒理解。

优先检查：
1. 短圆管长度 `76mm` 是否足够做几何导向
2. 三电机嵌入口袋是否互相干涉
3. 外转子旋转包络与 saddle / 管壁间隙
4. 45×45 单端固定与主框架接口
5. `+Z` 出口轴线与发射方向是否满足总装要求

v1.2a 是主线收紧版：
- 短圆管 / 短环中心
- 三外转子嵌入夹持
- 以圆管出口为发射方向
- 保留 45×45 单端安装
- 为后续几何干涉、运动包络、50m/RPM、180° 弧线控制仿真做基准
