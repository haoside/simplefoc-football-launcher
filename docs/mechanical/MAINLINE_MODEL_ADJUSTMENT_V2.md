# MAINLINE_MODEL_ADJUSTMENT_V2

## 目标
把当前 6354 主线模型从“几何占位”推进到“工程并线可评审”状态，并吸收 `outrunner_launch_tube_v1_1` 的接口信息。

当前并线基准：
- 主线模型：`cad/assembly/p0-single-ball-launcher-v2/p0_single_ball_launcher_outer_rotor_parametric_v10.scad`
- 发射筒版本：`cad/assembly/p0-single-ball-launcher-v3/outrunner_launch_tube_v1_1.scad`

---

## 并线检查结论

### 1. 几何主线一致，可并线
两边当前几何口径一致：
- 6354 外转子
- 三电机中心线同平面
- 三工位等边三角形 / 120° 均布
- 3 外转子与球直接接触
- 底面固定逻辑成立
- 可调，但刚性优先

### 2. 需要显式对齐的参数
当前应统一按下列参数做工程评审：

| 参数 | 主线当前 | v1.1 当前 | 结论 |
|---|---:|---:|---|
| `ball_d` | 220 | 220 | 一致 |
| `motor_d` | 63 | 63 | 一致 |
| `motor_len` | 53.5 / 6354 占位 | 54 | 统一按 54 占位 |
| `inner_d` | `ball_d + ring_clearance = 236` | 232 | **优先按 232 收口** |
| `preload/contact_sink` | preload = 4 | 2–3，默认 2.5 | **主线需改为 2.5 默认，2–3 可调** |
| 固定方式 | bottom-face mount | end-face bridge + base-side adjust expression | 需保留“底面固定优先”口径，同时把近底座微调表达补进主线 |

### 3. 当前关键差异
主线 v10 仍偏“总装关系表达”，不足以覆盖 v1.1 的以下信息：
- `inner_d=232` 的发射筒边界
- 2–3mm 压紧量的可调区间
- 端面槽孔化安装孔位表达
- 三侧透明防护罩安装孔位
- 开窗圆角 / 内侧 relief
- 近固定端的小行程调节结构表达

因此当前建议不是简单替换，而是：
**v1.1 作为发射筒与电机接口子模型基线，主线总装继续保留 v10，但参数与接口说明必须同步。**

## 主线模型调整动作

### A. 立即同步的参数
建议主线模型下一版至少同步：

```scad
motor_series = "6354";
motor_body_d = 63;
motor_body_len = 54;      // 不再用 53.5 占位
ring_clearance = 12;      // 对齐 inner_d = 232
preload = 2.5;            // 作为默认压紧量
```

并在注释中明确：
- 压紧量允许 2–3mm 调整
- 包胶厚度变化会直接改变最终压紧量
- 实测电机尺寸未锁前，孔距仍为参数占位

### B. 支撑路径强化
主线当前已有：
- 电机底面支撑座
- 双立柱
- 斜撑
- 三工位对称支撑

下一版建议继续强化：
- `mount_base_t`：继续加厚，优先表达底部支撑座刚性
- `brace_t`：增加到更明显的工程表达级别
- 缩短 `brace_len`，让支撑链路更短
- 三工位保持完全同构，不允许一侧先特殊化

### C. 调节结构并线
把 v1.1 的“近固定端小行程调节”吸收到主线：
- 只允许近底座 / 近固定端做短槽调节
- 不允许在环边缘长悬臂调节
- 不允许调节件成为主承载件

推荐表达：
- 底座短槽 + 定位孔
- 垫片 / 分级孔位
- 螺栓组驱动的短行程微调

### D. 接口件信息补充
主线文档需要补充以下接口状态：
- 端面孔距未锁，保持参数化槽孔
- 包胶厚度未锁，需与压紧量联动校核
- 包胶后动平衡为单独验证项
- 透明防护罩需复核散热与装配空间

## 对设计师的接口反馈（v1.1a 仅在需要时修改）
当前先不要求大改，v1.1 可以保持冻结。

仅保留以下潜在 `v1.1a` 触发条件：

1. **主线确认继续使用 `inner_d=232`** 后，如发现：
   - 球道与窗口边界距离不足
   - 防护罩孔位与外框干涉
   - 包胶后压紧超过 3mm
   再做定点修正。

2. **6354 实测孔位出来后**，若与当前默认槽孔偏差过大：
   - 调整 `mount_hole_x_spacing`
   - 调整 `mount_hole_z_spacing`
   - 调整 `mount_slot_extra`

3. **防护罩与散热/维护冲突** 时：
   - 仅修正耳位位置或孔距
   - 不改变三窗口主几何

## 风险复核

### 必列验证项
1. 6354 实测外径 / 长度 / 端面孔距
2. 包胶厚度对压紧量的联动影响
3. 包胶后动平衡
4. 透明防护罩的散热 / 装配 / 维护空间
5. 端面固定板与主线外框的工具进入空间

### 当前阶段判断
- **几何正确**：可以
- **固定逻辑正确**：可以
- **可调逻辑正确**：可以，但必须限制在小行程
- **可直接进制造**：还不行
- **可进入 v1.2 工程验证版**：**可以，前提是把上面 5 个验证项显式带上**

---

## 结论
当前建议：

1. **接受设计师 v1.1 作为发射筒子模型冻结版**
2. **主线总装按 232 内径 / 54 长度 / 2.5mm 默认压紧量同步参数**
3. **保留参数化槽孔，不写死 6354 端面孔距**
4. **列清楚包胶、动平衡、防护罩三类风险**
5. 满足以上条件后，**可进入 `v1.2 工程验证版`**
