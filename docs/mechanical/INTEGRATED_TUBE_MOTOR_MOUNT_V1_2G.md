# Integrated Tube + Three Motor Mount V1.2G

## 当前口径
按 owner 参考图收紧：

**发射圆管兼三电机固定一体结构，必须贴近图片。**

结构不再理解为：
- 独立圆管 + 外挂电机板
- 圆管外另加保护罩
- 电机通过分离支架临时靠近球道

当前主线理解为：

```text
中心短发射管 + 三个周向电机嵌入口袋 + 端面压板 + 加强筋 = 一个橙色一体夹持件
```

---

## 一体结构组成

### 1. 中心短发射管
- 保持短管 / 短环口径
- 中心开口作为球道
- 管壁外侧直接长出三电机口袋

### 2. 三电机嵌入口袋
- 3 个 6354 外转子沿圆管周向约 120° 分布
- 口袋与圆管外壁连续成型
- 内侧保留接触窗口，让外转子直接接触足球
- 电机不是外挂在外圈上

### 3. 端面压板
- 每个电机口袋两端有压板 / 端盖限位
- 端面螺丝孔用于固定电机或压紧件
- 端面压板并入一体橙色结构，不单独漂浮

### 4. 加强筋
- 圆管与电机口袋之间增加三角 / 矩形加强筋
- 加强筋负责抵抗电机压紧力、振动和侧旋反扭矩
- 加强筋布置需避开线束出口和工具进入空间

## 工程待验证

1. **口袋互相干涉**
   - 三个电机口袋在 roll 90° 后是否互相碰撞

2. **转子包络**
   - 外转子旋转时与口袋、端压板、加强筋的最小间隙

3. **接触窗口**
   - 内侧窗口是否足够让包胶转子压到球面
   - 是否存在管壁遮挡接触带

4. **端面固定**
   - 端压板孔位能否接真实 6354 端面
   - M5 / M4 螺丝工具是否可进入

5. **线束出口**
   - 三电机线束在一体口袋中不能被夹死
   - 需要预留线槽或后侧缺口

6. **强度与制造**
   - 一体打印/加工件是否需要分件拆装
   - 加强筋是否影响装配顺序
