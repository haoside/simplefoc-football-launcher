## 0. 电磁弹射（EML）候选类型

| 类型 | 原理 | 足球适配性 |
|------|------|-----------|
| 线圈炮（Coilgun / Gauss） | 多级同步线圈磁场加速 | ❌ 极难，足球质量 100× 典型弹丸 |
| 直线感应电机（LIM） | 行波磁场推力 | ⚠ 可行但效率低、控制复杂 |
| 轨道炮（Railgun） | 洛伦兹力直接推 | ❌ 极危险、磨损大、不适合球 |
| 同步直线电机（PMLSM） | 永磁+线圈同步 | ⚠ 可行但线圈复杂 |

## 1. 能量需求估算

球目标：220mm, 430g, 50m 射程 → 出膛速度 v ≈ 25 m/s

```
KE = 0.5 × m × v²
   = 0.5 × 0.43 × 25²
   = 134 J  ← 球获得的动能
```

**EML 系统总能耗**：
- 线圈炮典型效率 1-5%
- 输入能量 = 134 / 0.02 ≈ **6700 J**
- LIM/PMLSM 效率 30-60%
- 输入能量 ≈ **225-450 J**

**对比气压（基准）**：
- 8 bar / 20 L 气罐内能 = P × V = 8e5 × 0.020 = **16000 J**（理论）
- 实际有效做功 ≈ 5-15%（阀门 + 摩擦 + 漏气）
- 球获得 ≈ 800-2400 J 等效，远超 134 J → 8 bar 完全够用

## 2. EML 用于足球的具体难点

### 2.1 质量 / 尺寸失配
- 标准线圈炮弹丸：1-50 g
- 标准 5 号足球：430 g（**8-400× 重**）
- 足球直径 220 mm 需要巨大的线圈孔径
- 线圈电感 L 随面积增大 → 电流上升慢 → 加速力下降
- 结论：**同等速度下需要 8-400× 的磁场能量**

### 2.2 电容 / 储能
- 6700 J 电容储能 ≈ 1-2 kg 大电容（400V, 几十 mF）
- 充电时间数秒 → 不能快速连发
- 高电压（>300V）危险

### 2.3 安全
- 电容放电瞬间电流 >10 kA
- 强磁场对心脏起搏器、磁性物危险
- 线圈炮失败模式（弹丸卡住、放电短路）→ 烧毁

### 2.4 磨损
- 足球不像金属弹丸，无导轨约束
- 球在管内无固定电磁锚点，会被磁场反弹/偏转
- 必须有机械导轨（限制电磁优势）

### 2.5 控制
- 球质量大、运动慢 → 需要长加速段（>500mm）
- 多级线圈需要精确同步触发（μs 级）
- 闭环反馈（球位置/速度）极难

## 3. 与气压方案对比

| 维度 | 电磁弹射 | 压缩空气 |
|------|---------|---------|
| 球 25 m/s 所需储能 | 6700 J（输入） | 16000 J（理论）/ 200-500 J（有效） |
| 设备复杂度 | 高（电容 + 线圈 + 触发） | 中（罐 + 阀 + 管） |
| 安全等级 | **高危**（>300V, kA 级电流） | **中危**（<8 bar, 标准件） |
| 成本 | 高（电容/线圈/IGBT） | 低（标准气动件） |
| 维护 | 难（电气） | 易（机械） |
| 连续发射 | 受电容充电限制 | 受储气量限制 |
| 旋转控制 | 可调线圈时序 | 单独电机模块 |
| 商业参考 | 极少（仅电磁炮爱好）| **多**（FORZA / Jug's 等）|
| 50m 目标可行性 | 可行但极端 | ✅ 已验证可行 |

---

## 4. 结论

### 4.1 电磁弹射 P0 不推荐

理由：
- **能**在原理上替代气压
- 但**成本、安全、维护、复杂度**全面劣于气压
- 仅在**没有气源 / 完全电磁化**的特殊场景下有意义
- P0（首版 / 训练场 50m 验证）首选仍是气压

### 4.2 何时考虑 EML

- 远期：电磁化发射阵列（无气源环境、太空、月球基地）
- 玩具/小型球：低压线圈炮（<1J 储能）
- 教学演示：展示 Lorentz 力的应用

### 4.3 折中方案：LIM 短距离辅助

如果未来想做"无气源"版本：
- LIM 段 + 电磁预加速
- LIM 仅作短距离（<100mm）辅助
- 主推进仍靠气压或弹簧储能

## 5. 仿真口径（如果以后要做 EML）

```
EML 输入:
- 线圈匝数 N
- 线圈电流 I(t)
- 加速段长度 L_accel
- 球质量 m

EML 输出:
- 出膛速度 v_exit
- 加速时间 t_accel
- 储能 E_stored
- 系统效率 η
```

参考开源工具：openEMS（FDTD 电磁仿真）、FEMM（2D 磁场有限元）
