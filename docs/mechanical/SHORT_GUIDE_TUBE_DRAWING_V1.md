# 短导向筒 3D 打印图纸 v1

## 交付文件
- `cad/3d-print/short-guide-tube-v1/short_guide_tube_v1.scad`
- `cad/3d-print/short-guide-tube-v1/README.md`

## 当前尺寸
- 长度：200mm
- 内径：232mm
- 壁厚：4mm
- 喇叭口长度：35mm
- 法兰外径：270mm
- 安装孔：6 x M5
- 孔圆直径（PCD）：252mm

## 设计边界
- 仅作短导向 / 初始方向稳定
- 不是供球管 / 长管膛
- 必须可拆卸
- 必须用于 A/B 测试：无导向 / 150 / 200 / 250mm

## 打印建议
- 材料：PETG 或尼龙优先
- 打印机尺寸不足时，拆成两片半壳再法兰拼接
- 内壁优先打磨/抛光，降低摩擦
