# 机械装配与试验 v1

## 装配顺序
1. 底座与主框架
2. 电机安装板
3. 电机与轮组
4. 控制仓 / 电源仓
5. 线束与急停
6. 手动上球导向口
7. 三轮位置粗调
8. 防护罩安装
9. 静态联调

## 装配检查点
- 底座刚性
- 主框架对称性
- 电机安装板调节顺畅
- 轮组偏摆可接受
- 联轴器安装无明显偏心
- 动力线束与信号线分离
- 急停硬切有效
- 手动上球导向顺畅
- 三轮接触大致均匀
- 防护罩不干涉轮组

## 三轮初始调节
- wheel1 @ 12:00
- wheel2 @ 4:00
- wheel3 @ 8:00
- 三轮都先退开，以球为基准逐个靠近
- 上轮先定，再调下两轮对称
- 初始预压建议三轮各 3mm
- 总预压缩量先落在 5~10mm

## 轮面/轮径试验
### 轮面材料
- PU 80A（默认）
- PU 70A
- PU 90A
- 硅胶 70A

### 轮径
- 100mm
- 110mm（默认）
- 120mm

## 第一轮机械试验组合
- G1：110 + PU80A + 3/3/3 mm
- G2：110 + PU80A + 4/4/4 mm
- G3：110 + PU70A + 3/3/3 mm
- G4：120 + PU80A + 3/3/3 mm

## 第二轮机械试验（加入旋转）
- R1：STRAIGHT
- R2：TOPSPIN
- R3：LEFT_CURVE
- R4：更大预压 + TOPSPIN

## 第三轮联合试验
- 机械 + 电控 + 安全联合验证
- 直射 / 上旋 / 左右偏 / 急停 / fault / 连续发射
