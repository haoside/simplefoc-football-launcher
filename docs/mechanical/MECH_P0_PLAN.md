# P0 机械结构设计方案（单球 3 轮摩擦轮）

## 冻结前提
- 单球手动上球
- 3 个摩擦轮，120° 均布
- wheel1 @ 12:00
- wheel2 @ 4:00
- wheel3 @ 8:00
- 标准 5 号足球，直径约 220mm，质量约 430g
- 无连续供弹、无球仓、无闸门

## P0 机械目标
1. 稳定夹持 5 号球
2. 通过 3 轮摩擦加速完成射出
3. 三轮相对球心位置可调
4. 结构具备防护罩与急停安装位
5. 便于轮面、预压量、轮距的快速试错

## 机械模块拆分
### 1. 轮组模块
- 三个摩擦轮
- 三个电机安装座
- 三个径向可调滑槽
- 轮轴 / 联轴器 / 轴承支撑

### 2. 主框架模块
- 底座
- 主安装环 / 三角支撑架
- 电机安装板
- 控制板安装位
- 电源安装位

### 3. 上球导向模块
- 单球手动放球口
- 导向喇叭口 / 导向环
- 进入轮组前的定位支撑面

### 4. 防护模块
- 透明 PC 防护罩
- 前部出球窗口
- 侧面检修开口
- 急停按钮安装位

## 关键设计原则
### A. 可调优先
- 三个轮位必须可调
- 预压量必须可调
- 轮面材料必须可替换

### B. 刚性优先
- 三轮相对位置不能在加载时明显漂移
- 电机安装板和主框架优先铝板/型材，不先追轻量化

### C. 保护优先
- 发射区必须有罩
- 轮组外露最小化
- 急停位于右手易触达区
