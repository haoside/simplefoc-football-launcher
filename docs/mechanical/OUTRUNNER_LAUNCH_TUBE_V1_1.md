# OUTRUNNER_LAUNCH_TUBE_V1_1

## 目标
在 `outrunner_launch_tube_v1` 基础上收口到 **6354 外转子版本**，保持冻结约束不变：
- 三电机中心线同平面
- 三工位等边三角形分布
- 3 外转子工作带与球相切
- 底面固定逻辑优先
- 可调，但刚性优先于大行程
- 不回退到星型/Y 型汇聚、三轴平行、独立摩擦轮、顶面主安装

对应 CAD：
- `cad/assembly/p0-single-ball-launcher-v3/outrunner_launch_tube_v1_1.scad`

---

## v1 → v1.1 变更点

### 1. 电机规格从 6374 调整为 6354
默认参数改为：
```scad
motor_d   = 63;
motor_len = 54;
```
说明：
- 保留参数化，不写死真实孔位与外壳尺寸
- 发射筒 `inner_d=232` 暂维持不变
- 压紧量表达改为 **2–3mm 区间**，默认 `2.5mm`

### 2. 端面固定孔位参数化
v1 只有占位孔距；v1.1 改成更明确的参数化适配板表达：
```scad
mount_hole_d         = 5.5;
mount_hole_x_spacing = 30;
mount_hole_z_spacing = 30;
mount_slot_extra     = 4;
```
含义：
- 当前仍是待实测前的默认值
- 通过槽孔而不是纯圆孔，给 6354 实测孔距保留修正余量
- 便于后续转成适配板模板而不推翻整体结构

### 3. 发射筒开窗圆角 / 倒角表达
v1.1 对三侧开窗做了两层处理：
- 外轮廓显式圆角：`window_corner_r = 10`
- 内侧入口 relief：`window_inner_relief = 6`

目的：
- 降低窗口开口对球皮的切削风险
- 降低开窗后的应力集中
- 给包胶/软包边预留工艺余量

### 4. 三侧透明防护罩安装孔表达
新增每个窗口四点式安装耳位：
- `shield_hole_d = 4.5`
- 每侧窗口 4 个孔位
- 逻辑上用于 PC/透明罩板固定

作用：
- 约束异物卷入
- 给高速外转子提供基本防护边界
- 后续可扩展透明罩、网罩或局部挡片

### 5. 筒身加强肋优化
在 v1 外框肋基础上继续补强：
- 保留窗口周边框式加强肋
- 额外增加窗口上下两条纵向加强肋
- 保持三工位对称，避免单边刚度漂移

目的：
- 减少开窗后筒身局部塌陷
- 让固定与受力路径更连续
- 给防护罩安装耳提供更稳定的边界刚度

### 6. 压紧量 2–3mm 可调表达
新增：
```scad
contact_sink      = 2.5;
contact_sink_min  = 2.0;
contact_sink_max  = 3.0;
```
并增加近固定端的短行程调节槽表达：
- `adjust_slot_len = 18`
- `adjust_slot_w = 6`

说明：
- 这里表达的是 **小范围校准**，不是大行程滑台
- 目标是装配校正与包胶厚度补偿
- 仍坚持“刚性优先，不追求大范围可调”

## 关键结构逻辑

### 1. 受力关系
- 球被三只 6354 外转子从三侧直接夹持 / 驱动
- 接触面仍是外转子壳体，不引入独立摩擦轮
- 电机通过端面固定板与桥接件固定在筒外
- 载荷先进入端面固定板，再进入桥接结构与外围安装体系

### 2. 开窗关系
- 三个窗口按 120° 均布
- 窗口尺寸由 `motor_d`、`motor_len` 自动带出
- 保持球道中心不变，避免回到三轴平行布局

### 3. 调节关系
- 调节只发生在靠固定端附近
- 主要用于：
  - 6354 实物尺寸差异补偿
  - 包胶厚度补偿
  - 压紧量从 2mm 到 3mm 微调

## 结构风险

### 风险 1：6354 实测尺寸偏差可能影响窗口包络
如果真实 6354 外壳直径、长度与默认值偏差较大：
- 会影响窗口大小
- 会影响接触带位置
- 会影响防护罩孔位边界距离

### 风险 2：端面孔距未实测前只能视为占位模板
当前 CAD 已参数化，但默认孔距不应直接下生产。
必须在实测后锁定：
- 孔距
- 中心凸台避让
- 紧固件规格
- 装配工具空间

### 风险 3：包胶后动平衡风险上升
如果外转子壳体需要包胶增摩：
- 动平衡要求会明显上升
- 高转速下振动可能放大
- 必须明确允许的胶层厚度与质量一致性

### 风险 4：开窗边缘仍需软处理
即使 CAD 已做圆角 / relief：
- 对真实足球皮革 / TPU 外皮仍可能偏硬
- 仍建议实物阶段增加软包边或可替换护条

### 风险 5：透明防护罩孔位存在二次干涉可能
透明罩加上后，需要复核：
- 电机壳体旋转包络
- 安装螺钉头空间
- 散热气流路径
- 人手上球动作空间
