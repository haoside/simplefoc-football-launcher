# 弧线旋转单元 · 最终结构设计

> 最后更新：2026-07-03
> 状态：**✅ 结构锁定**

## 结构原理

一体发射筒 + 3 个径向支架臂（垂直于筒轴），120° 均布。每个臂尖固定一个 6374 BLDC 电机，外转子壳面通过筒壁窗口伸出接触球面。三电机旋转方向一致，产生球的正旋（绕行进轴）。

## 关键参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 发射筒 | ID Φ220 × L300mm | 一体式，壁上开 3 个窗口 |
| 支架臂 | 垂直于筒轴 | 一体成型（筒+臂=同一零件） |
| 电机 | 6374 BLDC 140KV × 3 | 外转子直驱 |
| 电机固定 | 圆形端面 4×M4（间距45mm） | 固定到臂尖 |
| 转轴支撑 | 臂端轴承过孔 | 不接触旋转件 |
| 旋转方向 | 三电机一致 | 同速=正旋直线球，差速=弧线球 |
| 壳面 | Φ63mm 光面 | 通过窗口伸出接触球面 |

## 电机安装方式

1. 支架臂从一体筒径向伸出（⊥筒轴）
2. 电机圆形端面朝向臂尖，4×M4 螺丝固定
3. 外转子圆柱面朝筒心，通过窗口伸出
4. 壳面旋转方向平行筒轴，接触球面产生正旋
5. 转轴从电机另一端穿出，由臂端轴承过孔支撑
