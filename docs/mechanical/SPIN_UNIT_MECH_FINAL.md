# 弧线旋转单元 · 最终结构设计

> 最后更新：2026-07-10
> 状态：✅ 结构锁定

## 结构原理

一体发射筒 + 3 个 6374 BLDC 电机围绕筒壁，120° 均布。电机轴延长线构成等边三角形，球在三角形内切位置。外转子圆筒面通过筒壁窗口接触球面。三电机旋转方向一致，产生球的正旋。

## 几何关系（核心约束）

- 三个电机轴指向筒心方向（径向内指），但轴线不过筒心
- 每根轴的延长线是等边三角形的一条边
- 球位于等边三角形的内切圆位置
- 外转子圆筒面（电机最大旋转面）接触球面
- 电机端面朝外，固定到支架

## 关键参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 发射筒 | ID Φ220 × L300mm | 一体式，壁上开 3 个窗口 |
| 电机位置 | 靠近出口端 | 球先过通道，出口附近加旋 |
| 电机 | 6374 BLDC 140KV × 3 | 外转子直驱 |
| 电机固定 | 圆形端面 4(8) 螺孔 | 端面朝外，固定到支架 |
| 旋转方向 | 三电机一致 | 同速=正旋直线球，差速=弧线球 |
| 壳面 | 外转子圆筒面 | 电机最大旋转面，接触球面 |
| 轴方向 | 指向筒心 | 延长线构成等边三角形 |

## 电机安装方式

1. 电机轴指向筒心方向（径向内指）
2. 外转子圆筒面通过筒壁窗口伸出，接触球面
3. 电机圆形端面朝外，固定到支架
4. 三电机 120° 均布，轴延长线构成等边三角形
5. 三电机同向旋转，球被加旋
