# 弧线旋转单元 · 最终结构设计

> 最后更新：2026-07-11
> 状态：✅ 结构锁定

## 结构原理（三角结构）

三个 6374 BLDC 电机位于等边三角形的三个顶点。每个电机的轴延长线指向下一个电机方向，三根轴延长线 = 等边三角形的三条边。发射筒位于三角形中心，电机轴穿过筒体（3D 圆筒）。外转子圆筒面通过筒壁窗口接触球面。

## 几何关系（核心约束）

- 三电机位于等边三角形**三个顶点**
- 每个电机轴延长线指向下一个电机 → **三根轴 = 三角形三条边**
- 发射筒在三角形**中心**
- 轴穿过筒体（3D 圆筒，不是仅截面圆）
- 外转子圆筒面（电机最大旋转面）接触球面
- 三电机旋转方向一致

## 关键参数

| 参数 | 数值 | 说明 |
|------|------|------|
| 发射筒 | ID Φ220 × L300mm | 一体式，壁上开 3 个窗口 |
| 电机位置 | 靠近出口端 | 球先过通道，出口附近加旋 |
| 电机 | 6374 BLDC 140KV × 3 | 外转子直驱 |
| 电机布局 | 等边三角形三顶点 | 轴延长线 = 三角形边 |
| 电机固定 | 圆形端面 4(8) 螺孔 | 端面朝外固定到支架 |
| 旋转方向 | 三电机一致 | 同速=正旋，差速=弧线 |
| 壳面 | 外转子圆筒面 | 电机最大旋转面，接触球面 |
