# 外转子直驱发射筒 v1

## 设计结论

采用“端面固定电机 + 发射筒开窗 + 外转子直接接触足球”的发射头方案。

本方案不使用轴接额外摩擦轮；6374 / 63xx 外转子壳体本身作为接触驱动面。电机轴仅视为内部支承，不作为外露驱动件。

- CAD：`cad/assembly/p0-single-ball-launcher-v3/outrunner_launch_tube_v1.scad`

## 配色 / 结构语义

- 粉色：发射筒、固定架、端面固定板
- 蓝色：筒身开口 / 外转子穿窗区域
- 白色：外转子旋转壳体
- 黄色：足球与外转子的压紧接触带

## 布局

- 发射筒内径：`232 mm` 起步，匹配 5 号球 `220 mm` 名义直径并保留导向余量
- 筒长：`260 mm` 起步
- 前后喇叭口：`35 mm`，用于手动上球与出球容差
- 三个筒身开口：`120°` 均布
- 三颗电机轴线沿发射筒切线方向布置
- 外转子壳体穿过筒身开口，直接压紧足球表面

## 关键元素

1. **中心发射筒**
   - 负责球路导向与安全边界
   - 筒身开口处增加外部加强肋，降低开窗后的变形

2. **三处开窗**
   - 开口尺寸由 `window_z_len` 与 `window_tangent_w` 控制
   - 默认按 `6374` 外转子壳体尺寸预留
   - 开口边缘必须倒角或软包，避免刮伤足球

3. **端面固定模块**
   - 每颗电机通过左右端面板/适配板固定
   - 端面板之间用背桥连接
   - 中间区域避让外转子旋转壳体
   - 端面孔位当前为占位，必须按实际电机/适配板修正

4. **直接接触驱动面**
   - 外转子壳体为主驱动表面
   - 建议增加 `1–2 mm` 可更换高摩擦胶套/热缩套
   - 包胶后必须做动平衡检查

## 设计原则

- 不画、不采购、不实现轴接额外摩擦轮
- 支架固定电机端面，避开旋转外壳
- 外转子旋转外圆周向至少预留 `3 mm` 安全间隙
- 压紧量从 `2–3 mm` 低速验证开始
- 三个开窗外侧必须增加透明防护罩
- P0 阶段只做低速单球测试

## 待复核

- 实际 6374 / 63xx 外转子壳体直径、长度
- 端面固定孔距 / 适配板孔距
- 包胶后的外径、同心度、动平衡
- 开窗边缘是否刮球
- 发射筒材料与壁厚是否足够抗冲击
- 外转子壳体散热路径
