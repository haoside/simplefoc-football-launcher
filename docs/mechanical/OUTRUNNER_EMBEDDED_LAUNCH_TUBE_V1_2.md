## v1.2 结构定义

### 1. 发射圆管作为主结构
v1.2 不再把中心环作为外观主体，而是采用：
- 短发射圆管
- 管壁三处嵌入式电机窗口
- 管体外侧一体加强肋
- 单端 45×45 主安装接口

### 2. 三外转子嵌入管壁
每个电机站位包括：
- 管壁窗口
- 嵌入式电机口袋
- 背部 saddle 支撑
- 单侧端面固定夹板
- 外转子壳体伸入球道 `2–3mm`

目标：
- 不做独立摩擦轮
- 不靠轴驱动球
- 直接用外转子壳体 / 包胶层接触球

### 3. 电机翻转 180°
CAD 中加入：
```scad
motor_flip_180 = true;
```
含义：
- 固定端/端夹方向按参考图翻转
- 后续如果工程端确认线束/端面方向相反，只改这个参数即可

### 4. 单端 45×45 固定接口
继续锁定：
```scad
main_mount_pitch  = 45;
main_mount_hole_d = 5.5;
```
主安装板仍是一端固定，便于接入总装外框。

## 性能目标拆解

### A. 距离加强到 50m
50m 不是外观参数，是系统性能目标，需要验证：

1. 外转子有效线速度
2. 球与外转子接触时间
3. 包胶摩擦系数
4. 压紧量 2–3mm 是否足够传递能量
5. 球体变形与能量损耗
6. 三电机转速同步误差
7. 出口角度与初速度

#### 初步工程判断
- 结构上必须预留更高转速与更高夹持刚性
- 包胶后动平衡必须前置验证
- 50m 目标不建议直接跳制造，应先做仿真 + 台架测速

### B. 发射弧线 180° 可控
“180° 可控”应拆成控制目标：

1. 水平/俯仰出射角可调
2. 三电机差速可控
3. 通过上下/左右转速差生成旋转与弧线
4. 出球角度、初速度、球旋转速度可测

#### 设计含义
- CAD 只表达结构空间
- 真正的 180° 弧线控制来自：
  - 三电机独立转速控制
  - 可调安装角/整体发射角
  - 控制算法与测速反馈

## 3D 仿真路线：由简入繁，多轮更新

### Step 1：几何干涉仿真
目标：确认能装进去。
- 三个 6354 与发射圆管是否干涉
- 管壁窗口是否覆盖外转子工作带
- 45×45 安装板与外框是否冲突
- 防护罩/线束/工具空间是否够

输出：
- 干涉截图
- 最小间隙表
- v1.2a 修改清单

### Step 2：运动包络仿真
目标：确认转得起来。
- 外转子旋转包络
- 包胶后最大外径
- 防护罩安全间隙
- 球通过时的最小通道

输出：
- 旋转包络图
- 球路剖面图
- 风险点标注

### Step 3：简化动力学仿真
目标：估算 50m 所需初速度与转速区间。
- 球质量
- 出射角
- 空气阻力简化
- 目标距离 50m
- 初速反推

输出：
- 初速度范围
- 电机表面线速度范围
- 推荐 RPM 区间

### Step 4：接触/摩擦仿真
目标：验证能量能不能传给球。
- 包胶摩擦系数
- 压紧量 2/2.5/3mm 扫描
- 接触时间
- 球体变形
- 三轮同步误差

输出：
- 压紧量推荐值
- 包胶厚度建议
- 是否需要提高筒长/接触弧长

### Step 5：弧线控制仿真
目标：验证 180° 可控范围。
- 三电机差速模型
- 出球自旋方向
- 左右/上下弧线趋势
- 控制查表

输出：
- 电机转速组合表
- 预期轨迹族
- 控制边界

## 当前 v1.2 风险

### 1. 嵌入管壁后散热更差
电机被管壁包围后：
- 热堆积更明显
- 需要开散热槽或导风罩
- 防护罩不能完全封死

### 2. 50m 目标可能要求高转速
高转速带来：
- 包胶动平衡风险
- 轴承负载上升
- 外转子壳体安全风险

### 3. 2–3mm 压紧量可能不足或过大
要靠实测判断：
- 不足：打滑，距离不够
- 过大：球损伤、电机负载过高、堵球

### 4. 180° 弧线控制不是纯结构问题
需要电控与传感配合：
- 三电机独立闭环
- 出球测速
- 转速差控制
- 发射角机械调节
