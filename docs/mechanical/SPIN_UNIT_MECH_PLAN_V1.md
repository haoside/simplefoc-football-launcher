# 弧线旋转单元 · 机械设计方案 v1

> 最后更新：2026-07-02
> 架构 pivot：动力与旋转解耦，本方案仅覆盖弧线旋转单元

## 设计输入

- 电机：6374 BLDC 140KV × 3，外转子直驱
- 布局：120° 均布，电机轴垂直于筒轴（径向安装）
- 筒内径：Φ220mm（标准 5 号球通过）
- 功能：三电机独立调速，同速=正旋，差速=弧线

## 关键尺寸

| 参数 | 数值 | 说明 |
|------|------|------|
| 发射筒内径 | Φ220mm | 5 号球直径 216-226mm |
| 发射筒外径 | Φ290mm | 含壁厚 |
| 发射筒长度 | 300mm | 含电机嵌入段 |
| 壁厚 | 待计算 | 需承受电机扭矩+球面接触力 |
| 外转子壳面 | Φ63 × 150mm | 嵌入筒壁，接触球面 |
| 窗口尺寸 | Φ70mm | 壳面伸入通道，含装配间隙 |
| 电机轴径 | 按6374规格 | 径向安装，垂直于筒轴 |

## 结构原理

1. 发射筒水平放置，球从一端进入
2. 三个电机径向安装在筒壁外侧
3. 外转子壳面通过筒壁窗口伸入筒内
4. 壳面旋转时，接触点切向速度平行于筒轴
5. 三电机同速 → 球正旋（直线球）
6. 三电机差速 → 球弧线
