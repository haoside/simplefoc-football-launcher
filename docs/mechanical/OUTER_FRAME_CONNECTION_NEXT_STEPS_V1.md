# Outer Frame Connection Next Steps v1

## Current state

The launcher-head concept now includes the following chain:

- ring web sector
- support ear
- frame stub
- link plate
- outer mount plate

This is enough to express the intended load path direction, but not yet enough to claim a fabrication-ready outer frame connection.

## Next steps

### 1. Replace floating outer mount plates with explicit side support plates

Current outer mount plates are concept targets. Next revision should:

- define a left/right side support plate or triangular side bracket
- place outer mount holes on those plates explicitly
- make the head plate relationship readable without requiring interpretation

### 2. Add nominal head-to-frame spacing

Current model shows connection direction but not a fully disciplined spacing scheme.
Next revision should define:

- ring outer diameter to side plate distance
- side plate thickness
- link plate offset
- nominal bolt stand-off / spacer region if needed

### 3. Decide plate logic

Choose one and keep it stable for at least one iteration:

- radial side plates
- tangential side plates
- triangular outer bracket plates

### 4. Keep 6354 as baseline while doing this

Do not reopen 6374 as the mainline while refining outer frame connection.
Use 6354 as the packaging baseline; only compare 6360 / 6374 when a specific interference check requires it.
