# 电机安装板可制造图纸 v1

## 交付文件
- `cad/sheet-metal/motor-mount-plate-v1/motor_mount_plate_v1.scad`
- `cad/sheet-metal/motor-mount-plate-v1/README.md`

## 当前尺寸
- 外形：180 x 140 x 8 mm
- 中心轴孔：24 mm
- 电机安装孔：4 x 5.5 mm（暂定孔阵，待实物电机图复核）
- 主框架安装孔：4 x 6.0 mm
- 调节槽区域：120 x 20 mm

## 材料
- 6061 铝板，8 mm

## 设计原则
1. P0 优先可调，不先追轻量化
2. 电机孔位先做兼容口径，采购后按实物修正
3. 安装板应支持 30~40 mm 级的有效调节行程
4. 与主框架连接位优先做标准平板/CNC/水刀可加工形式

## 后续动作
- 采购到实际 6374 电机后，复核孔位与轴径
- 根据主框架真实结构，再定最终槽位方向与支撑孔位
