# P0 Assembly Model Notes v3

## Change summary

This version adds more real-machine structure to the OpenSCAD assembly placeholder:

- Head support ring around the 3-wheel launcher core.
- Three radial wheel/motor support arms.
- Short guide tube bracket.
- PC guard support rails.
- Better visual separation between the launcher head, chassis, battery bay, and operator side.

## Intent

Move the CAD export from a pure component layout toward a buildable prototype envelope.

## Still not final fabrication CAD

The following still need actual part/vendor drawings before fabrication freeze:

1. Real 6374 motor mounting drawing.
2. Real friction wheel CAD and rubber tread profile.
3. Actual aluminum profile sizes and fastening pattern.
4. PC guard flat pattern and bend/fastener details.
5. Battery/control bay actual enclosure drawings.

## P0 boundary reminder

- single-ball manual loading only
- short guide tube only, not feed tube
- no hopper / no gate / no continuous feed
