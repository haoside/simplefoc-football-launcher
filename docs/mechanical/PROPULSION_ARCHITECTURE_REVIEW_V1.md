## 0. 背景与触发

v1-v14 的多版外观迭代暴露一个事实：

> **单靠"三电机径向相切"模型，不能可靠把球沿圆管推出。**

不论把电机轴设为：
- 平行于管（v1-v5、v11-v13）→ 接触点切向速度无 Z 分量 → 球绕管自旋
- 径向垂直于管（v6-v10、v14 飞轮盘）→ Z 分量存在但**沿管切向相互抵消**

如果继续在外观层面迭代，无法锁定发射机理，因此停止 v15+ 外观，先做这份架构评审。

## 1. 球被推出圆管的必要条件

球（标准 5 号，220 mm，~430 g）受合力 F_net。沿管轴（Z 向）发射要求：

```
F_net_z > 0   且   F_net_z 持续时间足够（球加速到目标速度）
```

三种力源分别贡献：

| 力源 | 提供 F_z | 物理本质 |
|------|---------|---------|
| 气压差 | **强** | ΔP × A_ball |
| 摩擦轮切向 | 弱且易抵消 | μ × N × cos(切向) |
| 球自旋 Magnus | 0 沿轴 | 只影响横向飞行 |

**关键判定**：仅靠摩擦轮的"切向速度分量投影到 Z"在 120° 对称下平均为 0（除非破坏对称性或角度倾斜）。

## 2. 三个候选方案

### 方案 1：纯三外转子摩擦推进

**结构**：3×6374 电机径向相切，圆筒形 cradle，钢/橡胶 can 包覆电机外转子，球夹在三 can 之间。

**力学分析**：
- 接触点切向速度方向：v_t = ω_can × R_can
- Z 分量：v_z = v_t × sin(θ_contact)
- 120° 对称 + 同向旋转 → **Z 分量总和 = 0**
- 球只绕管自旋，不前进

**实测/仿真（参考 sim/football_launch_model.py 拓展）**：
- 球线速度 v_ball ≈ 0（仅自旋 ω_spin 可达数千 RPM）
- F_z 净力 ≈ 0
- **判定：球不能沿管发射**

**适用场景**：
- 球定位（预转 / 姿态稳定）
- 球旋转控制（弧线 / 香蕉球）
- **不能**作为唯一轴向推进来源

**复杂度**：高（3 电机 + 3 闭环 + 差速同步）
**风险**：烧电机、过流堵转
**验证路径**：仅做控旋测试

---

### 方案 2：压缩空气主推进 + 三电机控旋（推荐主线）

**结构**：
- 圆管后端封闭（压力端），接储气罐 + 快释放阀
- 球装载在管中预定位置（球挡或传感器触发）
- 触发：电磁阀瞬时开启 → 高压气体推动球 → 球被推出
- 三电机（径向相切）只负责：
  - 出膛前球姿态调整（预转 / 对中）
  - 出膛瞬间施加旋转（弧线球）
  - 修正出球稳定性

**力学分析**：
- 气压推力 F_air = (P_tank - P_atm) × A_ball
- 假设 P_tank = 8 bar (gauge) = 9 bar absolute
  - A_ball = π × (110)² = 38013 mm² ≈ 0.038 m²
  - F_air ≈ 8 × 10⁵ × 0.038 ≈ 30 kN（理论峰值）
  - 实际有效力受阀门流量、压力衰减影响，约 2-5 kN 持续 50-100 ms
- 球加速：a = F/m ≈ 5000 / 0.43 ≈ 12000 m/s²（瞬时）
- 球出膛速度 v_exit ≈ sqrt(2 × ΔP × A_ball × L_press / m) ≈ 15-25 m/s（典型 50 m 射程）

**判定**：球可沿管高速发射，方向、速度可控

**优势**：
- 球速度仅由 ΔP 控制，调节简单
- 三电机解脱为"控旋模块"，可独立设计验证
- 机械复杂度大幅降低
- 已有商业足球发射器采用此原理（FORZA、Net World Sports）

**风险**：
- 压力容器安全（PLA 长时间承压会疲劳 → 必须用 ABS/CF-PETG/铝合金）
- 必须有爆破片 + 泄压阀 + 急停
- 高压气管接口、电控阀同步

**复杂度**：中等（气路 + 阀 + 简单电控）
**验证路径**：
1. 5 bar 低压空气炮原理验证（PVC 管 + 球 + 阀门）
2. 测出膛速度 vs 压力曲线
3. 加入三电机预转测试
4. 加弧线球测试

---

### 方案 3：斜置 / 螺旋接触轮提供轴向分力

**结构**：电机径向**倾斜 5-15°** 于管轴（不严格垂直），或 can 表面刻螺旋槽（螺纹）。

**力学分析**：
- 接触点速度方向 v_t 不在径向平面内
- v_t 与管轴有夹角 → Z 分量 v_z ≠ 0
- 例如：can 倾斜 10°，v_z ≈ v_t × sin(10°) ≈ 0.17 × v_t
- 球速度较方案 1 大幅提升

**判定**：可发射，但倾斜机构复杂

**优势**：
- 仍是纯电机方案，无气压安全风险
- 单电机也能产生轴向分力（类比斜传送带）

**劣势**：
- 倾斜导致机械磨损不对称
- 球旋转控制变难（侧向分力耦合）
- 不适合 50 m 远射程（速度上限受摩擦功率限制）

**复杂度**：中（机械倾斜机构 + 螺旋 can 加工）
**验证路径**：单轮倾斜 10° 测试出膛速度

## 3. 三方案对比矩阵

| 维度 | 方案 1<br>纯摩擦 | 方案 2<br>气压+电机 | 方案 3<br>斜置/螺旋 |
|------|------|------|------|
| 球能沿管发射 | ❌ 不能 | ✅ 能 | ✅ 能 |
| 仅原地打转 | ✅ 是 | ❌ 否 | ❌ 否 |
| 适合 50 m 目标 | ❌ 否（速度 0） | ✅ 是 | ⚠ 受限 |
| 出膛速度调节 | 仅靠 RPM | **直接 ΔP** | 倾斜角 + RPM |
| 旋转控制 | 强（同向/差速） | 中（短时施加） | 弱（耦合） |
| 机械复杂度 | 高 | 中 | 中 |
| 安全风险 | 低 | **高（压力容器）** | 低 |
| 验证周期 | 短 | 中（需气源） | 中 |
| **推荐度** | 否定（仅作控旋） | **主线推荐** | 备选 |

## 4. 主线结论

### 4.1 球发射的力源分工（推荐方案 2）

| 力源 | 角色 | 量化目标 |
|------|------|---------|
| 压缩空气 ΔP | **主轴向推进** | 出膛 15-25 m/s，射程 30-50 m |
| 三电机径向 can | **控旋模块** | 自旋 0-2000 RPM 可调，弧线 ±5° |
| 重力 + 空气阻力 | 飞行段（不在本评审范围） | — |

### 4.2 三电机重新定位（不再做主推进）

电机的唯一职责：
- 球装载后、出膛前，对球预转
- 出膛瞬间可施加横向分力（弧线）
- 训练模式可纯电机低气压预转（无气）
- 故障保护（堵转 → 立即停气）

### 4.3 安全边界（气压方案必做项）

| 项目 | 要求 |
|------|------|
| 工作压力 | P0 ≤ 8 bar（gauge） |
| 爆破片 | 1.2× 工作压力标定 |
| 泄压阀 | 1.1× 工作压力开启 |
| 急停按钮 | 硬件 + 软件双切断 |
| 球装载互锁 | 球未到位禁止加压 |
| 发射方向 | 仅允许开阔室外、无人区 |
| 操作员距离 | ≥ 3 m |
| 屏蔽 | 球道 + 操作员之间物理隔板 |

### 4.4 验证路径（4 阶段）

**Stage 1：纯气炮原理验证（无电机）**
- 5 bar PVC 管 + 球 + 手动球阀
- 测出膛速度 vs ΔP 曲线
- 验证球确实沿管发射、速度可调
- **预计 1 周**

**Stage 2：加电磁阀 + 简单电控**
- 12V 电磁阀 + 按钮触发
- 加压力传感器 + 泄压
- 测重复性、一致性
- **预计 1-2 周**

**Stage 3：加三电机控旋模块**
- 现有 v9/v14 电机模块接入（径向）
- 测预转 → 触发 → 出膛的时序
- 测弧线球（差速预转 → 侧偏）
- **预计 2 周**

**Stage 4：整合 + 50 m 实地测试**
- 全部集成：气路 + 电控 + 三电机
- 安全联锁全开
- 50 m 射程标定
- **预计 2 周**

## 5. 仿真口径更新建议

需要更新 `sim/football_launch_model.py`：

### 新增"轴向推进来源"层

```python
# sim/football_launch_model.py 扩展

PROPULSION_SOURCES = {
    'friction_3rotor': {  # 方案 1
        'F_z_net': 0,           # Z 分量净力 = 0
        'ball_velocity': 0,     # 仅自旋
        'rotation': 'high',
    },
    'pneumatic_primary': {  # 方案 2
        'F_z_net': 'pressure_driven',
        'pressure_range_bar': (2, 8),
        'ball_velocity_range': (5, 25),  # m/s
        'rotation': 'motor_controlled',
    },
    'tilted_rotor': {       # 方案 3
        'F_z_net': 'tilt_angle_dependent',
        'tilt_range_deg': (5, 15),
        'ball_velocity_range': (3, 12),
        'rotation': 'partial_control',
    },
}
```

### 区分力源

| 力 | 公式 | 来源 |
|---|------|------|
| F_air | (P - P_atm) × A_ball | 气压 |
| F_friction_z | μ × N × cos(θ_contact) × sin(φ_tilt) | 摩擦轮（含倾斜补偿） |
| F_magnus | ρ × v × Γ × L | 球自旋（仅横向） |

### 输出字段

```
v_exit_z, v_exit_xy, ω_spin, ΔP, tilt_angle, F_z_net
```
