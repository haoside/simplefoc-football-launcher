# Mainline Model Adjustment Suggestions v1

## Goal

Translate the current mechanical priority into concrete model-adjustment guidance for the outer-rotor launcher head.

## Geometry premise

- 3 motor centerlines form an `equilateral triangle` in one plane
- 3 outer rotor shell working surfaces are tangent to the football
- bottom-face mounting only
- adjustability is required, but stiffness is the first priority

## Suggested adjustment order

### 1. Strengthen motor support interpretation first

Before adding more outer frame completeness, the mainline model should make each motor station read as a believable support node:

- thicker / more explicit motor bottom support base
- shorter apparent load path from motor to support base
- clearer brace geometry under each motor

### 2. Keep support symmetry consistent with equilateral centerline layout

Because the 3 centerlines are equilateral in one plane, the support logic should also read as balanced:

- equal brace logic on 3 stations
- equal support-base spacing logic
- equal adjustment intent on 3 stations

### 3. Introduce adjustment features near the base, not near the ring edge

Preferred first-pass adjustable mechanisms:

- short radial slots near the motor support base
- shim stacks / indexed step positions
- bolt-group-driven base adjustment

Avoid:

- long flexible outer tabs
- weak ring-edge-only adjustment arms

### 4. Delay cosmetic frame growth until support path is convincing

At this stage, every new line added to the model should serve one of these functions:

- fixation
- stiffness
- adjustment
- packaging clearance

Do not prioritize decorative frame completeness ahead of support clarity.
