# ENGINEERING_VALIDATION_V1_2

## 范围定义
`v1.2` 的目标不是开新方向，而是把当前冻结的 `v1.1` 方案整理成可验证、可制造、可改版的工程验证包。

当前冻结对象：
- `cad/assembly/p0-single-ball-launcher-v3/outrunner_launch_tube_v1_1.scad`
- `docs/mechanical/OUTRUNNER_LAUNCH_TUBE_V1_1.md`
- `docs/renders/OUTRUNNER_LAUNCH_TUBE_THREE_VIEW_V1_1.svg`

当前几何冻结前提：
- 6354 外转子
- 三电机中心线同平面
- 三工位等边三角形 / 120° 均布
- 3 外转子与球直接接触
- 底面固定优先
- 可调，但刚性优先
- 发射筒 `inner_d = 232`
- 默认压紧量 `2.5mm`，允许 `2–3mm` 微调

## 1. 6354 实测外径 / 长度 / 端面孔距

### 验证目的
把当前参数占位改成实物边界，避免主模型、窗口尺寸、安装孔位、防护罩孔位全都建立在假设值上。

### 验证方法
1. 取目标 6354 电机实物样件
2. 测量并记录：
   - 外壳实测直径
   - 外壳实测长度
   - 端面孔数 / 孔径 / 孔距
   - 中心凸台 / 轴承座直径
   - 紧固件规格与可用牙深
3. 与当前 CAD 参数逐项对照：
   - `motor_d`
   - `motor_len`
   - `mount_hole_x_spacing`
   - `mount_hole_z_spacing`
   - `end_face_center_hole_d`
4. 回填到 CAD 顶部参数并复核窗口包络

### 通过标准
- 实测数据已形成表格并归档
- CAD 参数已同步更新
- 槽孔补偿量仍在可接受范围内
- 电机端面安装不会碰到窗口边缘、防护罩耳位、桥接件

### 风险等级
**高**

### 失败后的修改路径
- 若孔距偏差小：仅修正槽孔参数，形成 `v1.1a`
- 若外壳长度/直径偏差明显：同步修正窗口尺寸、接触带位置、护罩孔位，形成 `v1.1a`
- 若端面结构与当前桥接逻辑不兼容：回到主线固定件设计，更新 `MAINLINE_MODEL_ADJUSTMENT`

## 2. 包胶厚度与压紧量联动

### 验证目的
确认包胶后总外径变化不会把默认 `2.5mm` 压紧量推到不可控区间。

### 验证方法
1. 设定包胶厚度候选值：
   - `0mm`
   - `1.0mm`
   - `1.5mm`
   - `2.0mm`
2. 计算每种包胶后的等效外径
3. 对照当前发射筒 `inner_d = 232` 与电机中心位置，计算最终压紧量
4. 复核调节槽是否足以把压紧量拉回 `2–3mm`
5. 输出联动表：
   - 包胶厚度
   - 等效外径
   - 理论压紧量
   - 是否需要调节

### 通过标准
- 对所有计划采用的包胶厚度，最终压紧量都能控制在 `2–3mm`
- 不需要为了补偿包胶去引入大行程柔性调节
- 小行程调节即可完成装配校准

### 风险等级
**高**

### 失败后的修改路径
- 若包胶后压紧量 > `3mm`：
  - 优先减薄包胶
  - 次选微调电机中心位置 / 调节槽范围
  - 必要时修正窗口和安装基准，形成 `v1.1a`
- 若包胶后压紧量 < `2mm`：
  - 通过垫片 / 调节槽补偿
  - 必要时增大包胶厚度规格下限

## 3. 包胶后动平衡

### 验证目的
确认外转子壳体加胶层后仍能在目标转速区间安全运行，不引入不可接受振动。

### 验证方法
1. 明确目标转速区间
2. 制作至少一组包胶样件
3. 做静平衡初筛：
   - 观察重心偏移
   - 检查胶层厚度一致性
4. 做低速到目标转速的逐级测试：
   - 振动
   - 异响
   - 温升
   - 固定件松动趋势
5. 记录：
   - 最大稳定转速
   - 振动是否可接受
   - 是否需要二次修整/配重

### 通过标准
- 目标工作转速区间内无明显异常振动
- 固定件不松动
- 包胶不脱落、不偏磨
- 不需要依赖临时性补胶修振动

### 风险等级
**高**

### 失败后的修改路径
- 改包胶工艺：材料、厚度、公差、贴附方式
- 降低允许最高转速
- 增加动平衡工序
- 若风险不可控，回退到“外转子工作带可替换套件”方案，不改几何主线但改工艺件定义

## 4. 透明防护罩散热 / 装配 / 维护空间

### 验证目的
确保防护罩不是“纸面安全件”，而是真能装、能拆、能散热、能维护。

### 验证方法
1. 基于当前护罩耳位建立透明罩包络
2. 检查：
   - 旋转壳体包络与护罩净距
   - 螺钉头与工具进入空间
   - 上球 / 维护动作空间
   - 散热气流路径
3. 做一次装配顺序推演：
   - 先装电机还是先装护罩
   - 护罩拆装是否需要大面积拆结构
4. 若有样件，做一次纸板/亚克力快速样件验证

### 通过标准
- 护罩可独立拆装
- 不碰转子包络
- 螺钉可实际拧到
- 不明显堵死散热路径
- 不显著妨碍上球与维护

### 风险等级
**中高**

### 失败后的修改路径
- 优先调耳位和孔距，形成 `v1.1a`
- 必要时改护罩分片方式
- 仍不改变三窗口主几何和 6354 主线关系

---

## 5. 端面固定板与外框工具进入空间

### 验证目的
确认当前固定结构不是“能画不能装”，尤其是螺丝、内六角、套筒和手部操作空间。

### 验证方法
1. 标出以下装配动作：
   - 安装端面固定板
   - 安装/调整桥接件
   - 固定到底座或外框
   - 调节压紧量
2. 为每个动作检查：
   - 工具进入方向
   - 工具摆角空间
   - 手部操作净空
   - 是否被护罩 / 窗口 / 外框遮挡
3. 输出装配顺序建议

### 通过标准
- 关键紧固件都有明确工具路径
- 不需要拆掉大半结构才能做压紧量微调
- 外框、护罩、端面固定板之间不存在硬干涉

### 风险等级
**中**

### 失败后的修改路径
- 调整螺钉方向
- 调整桥接件宽度/位置
- 调整外框与固定板间距
- 必要时改工具孔或检修窗口

---

## 验证执行顺序
建议按这个顺序走，避免返工：

1. **先锁 6354 实测尺寸**
2. **再做包胶厚度与压紧量联动**
3. **再做动平衡验证**
4. **再做透明护罩装配/散热验证**
5. **最后做工具进入与装配顺序验证**

原因：
- 前两项决定基础几何边界
- 动平衡决定外转子工艺是否可行
- 护罩和工具空间都属于后续装配边界验证

## 阶段结论口径
只有在以下条件同时满足时，才允许从 `v1.2` 进入下一阶段：
- 关键实测参数已锁
- 压紧量可控
- 包胶后可稳定运行
- 护罩可装可拆可维护
- 工具进入与装配顺序成立

否则，只允许回到：
- `v1.1a` 做定点修正
- 或更新主线固定/接口设计

不允许在 `v1.2` 阶段发散成新方向。
