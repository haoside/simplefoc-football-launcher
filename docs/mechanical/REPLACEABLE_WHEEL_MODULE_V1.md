# 可替换轮组模块 v1

## 背景

P0 仿真显示：

- `110mm + 3000rpm` 适合 P0-A 安全 bring-up。
- 标准弧线在 `110mm + 3000rpm` 下接近/触达 RPM 上限。
- `130mm + 4000rpm` 更适合作为 P0-B 现场调参候选。

因此机械设计不能只锁死 110mm 轮径，应支持 110/130mm 快速替换，减少后续返工。

## 冻结原则

1. **轮组可替换**：110mm 与 130mm 摩擦轮共用同一电机/轴/安装板基准。
2. **轮位可调**：径向调节范围至少 `±20mm`，覆盖不同轮径与预压缩量。
3. **预压缩可标定**：预压缩目标范围 `5–15mm`，安装结构要有刻度或定位孔。
4. **防护罩留余量**：PC 防护罩内包络按 130mm 轮径设计，110mm 通过调节补偿。
5. **安全优先**：轮组更换后必须重新跑空载、低速、急停验证。

## 推荐机械基线

| 项 | P0-A | P0-B 候选 |
|---|---:|---:|
| 轮径 | 110mm | 130mm |
| RPM cap | 3000 | 4000 |
| ramp | 5s | 6s |
| 用途 | 安全 bring-up | 现场弧线调参 |
| 防护要求 | 必须 | 必须加强复核 |

## 安装板要求

- 电机安装板保留长圆槽。
- 轮中心径向调节不小于 `40mm` 总行程。
- 轴承/联轴器区域按 130mm 轮最大包络校核。
- 轮组更换不应影响电机本体安装孔位。

## 需要工程复核

- 130mm 轮在 4000rpm 下的轮缘线速度。
- 轮胎材料强度与动平衡。
- 防护罩抗冲击能力。
- 电流峰值与 BMS/保险匹配。
