# P0 单球摩擦轮发射器尺寸草案

## 基础尺寸
- 足球直径：220mm（按 215~225mm 浮动设计）
- 轮径：110mm 起步，允许试验 100~120mm
- 轮宽：40~50mm

## 三轮布置
- wheel1：12 点钟
- wheel2：4 点钟
- wheel3：8 点钟
- 三轮围绕球心 120° 均布
- 每个轮位径向调节行程：±15mm
- 球预压缩量：5~15mm 可调

## 核心发射区包络
- 宽：500~600mm
- 高：450~550mm
- 深：250~350mm

## 电机安装位
按 6374 电机预留：
- 外径：65~70mm
- 长度：95~125mm
- 单电机安装板：180 x 140 x 8mm 起步
- 滑槽长度：30~40mm

## 整机建议尺寸（P0）
- 底座：1000 x 800mm 起步
- 整机宽：800~1000mm
- 整机深：700~900mm
- 整机高：900~1200mm

## 上球口
- 手动上球口宽度：230~260mm
- 喇叭口倒角/圆角导入，避免球先撞单轮
- 上球点到三轮中心的自由导入距离：100~180mm

## 防护罩
- 材料：6~8mm PC
- 轮组防护罩包络：
  - 宽 550~700mm
  - 高 500~650mm
  - 深 250~400mm

## 安装区域预留
- 控制板区：250 x 180 x 120mm
- 电源区：320 x 220 x 150mm
- 急停安装位：右前侧，无遮挡
