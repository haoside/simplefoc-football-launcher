# 多球管道 P0 机构拆分

## 目标
在 120° 三轮发射结构前增加可低频连续工作的多球管道供球机构，优先保证：
1. 单球可靠分离
2. 单次稳定到位
3. 发射后可回到下一球待发状态
4. 低频连续发射不卡球

## P0 默认方案
- 管道：立式或斜置多球管道
- 底部：单球分离机构（拨叉 / 闸门）
- 发射前段：缓冲导向段
- 维护：卡球检修开口

## 模块拆分
### 1. 球仓 / 管道
- 存储多颗标准 5 号足球
- 保证重力或辅助推送可持续下料
- 内壁需兼顾低摩擦与不易卡球

### 2. 单球分离机构
- 一次只允许一颗球进入待发位
- 防止双球同时进入轮组
- 需要有 fail-safe 默认位

### 3. 待发位 / 定位机构
- 让球在三轮前保持可重复位置
- 与 `BALL_IN_POSITION` 传感器联动

### 4. 导向段
- 降低供球冲击
- 避免球体偏斜进入轮组

### 5. 检修与清障
- 预留手动开口
- 便于排除双球叠入 / 卡球

## P0 通过标准
- 单球分离成功率高
- 连续多球低频发射无明显双球误入
- 检修动作简单，不需大拆结构
