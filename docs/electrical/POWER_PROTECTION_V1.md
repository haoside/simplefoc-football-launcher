# 电源保护与配电 v1

## 冻结参数
- 总保险：150A
- 每路支路保险：40A ×3
- BMS：持续 ≥150A，峰值 ≥250A
- 主动力线：6AWG
- 驱动支路 / 电机三相：10AWG
- 低压支路：16~18AWG
- Hall / CAN / 控制：22~24AWG
- 单路连续限流：20A
- 单路短时峰值：35A
- 升速策略：800~1500ms ramp，必要时错峰 100~200ms
- 急停：两级切断（先 EN，后主动力）

## 配电拓扑
```text
24V LiFePO4
  -> BMS
  -> 主断电开关
  -> 150A 总保险
  -> 主接触器 / 急停主切
  -> 主配电块
      -> 40A 支路保险 -> 驱动1
      -> 40A 支路保险 -> 驱动2
      -> 40A 支路保险 -> 驱动3
      -> 10A 支路保险 -> DCDC
```

## 急停链路
1. 一级：立即拉低 DRV1_EN / DRV2_EN / DRV3_EN
2. 二级：切断 24V 主动力母线

## 地线策略
- 动力地统一回主配电负极
- 控制地与动力地单点汇接
- Hall / CAN 地不要长距离乱串
