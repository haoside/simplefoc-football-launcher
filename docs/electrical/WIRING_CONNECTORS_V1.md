# 线束与连接器清单 v1

## 线径冻结
- 主动力：6AWG
- 驱动支路 / 电机三相：10AWG
- 低压支路：16~18AWG
- Hall / CAN / 控制：22~24AWG

## 动力线束
- 电池 -> 主开关 -> 总保险 -> 主接触器：6AWG
- 主配电块 -> 驱动1/2/3：10AWG
- 驱动 -> 电机三相：10AWG
- 主配电 -> DCDC：16~18AWG

## 信号线束
- Hall A/B/C：22~24AWG
- CAN 总线：22AWG 双绞线
- EN / FAULT / 控制线：22~24AWG
- 急停控制线：20~22AWG

## 连接器建议
### 动力侧
- 电池接口：XT90 / Anderson 级
- 主配电输出：螺丝端子 / 铜排压接端子
- 驱动到电机三相：大电流端子 / 螺丝端子

### 信号侧
- Hall：JST-XH / JST-VH
- CAN：JST-GH / 端子台
- 调试口：排针 / 小型接口
- 急停：锁紧式工业端子

## 布线原则
1. 动力线和信号线分开
2. 三相动力线尽量短
3. Hall 线远离三相线
4. CAN 走一条主干，不乱分叉
5. 线束固定在框架内侧，避免蹭轮组
