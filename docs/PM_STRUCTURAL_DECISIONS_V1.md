## 架构 Pivot（2026-07-02）

**原始方案**：3 电机摩擦发射（电机同时提供动力 + 旋转）
**新方案**：动力与旋转完全解耦

| 子系统 | 功能 | 状态 |
|--------|------|------|
| 弧线旋转单元 | 3×140KV 电机嵌入筒壁，专职提供旋转 | 本次设计范围 |
| 发射动力单元 | 压缩空气 / 电磁弹射 | 另行设计 |

## 弧线单元参数 ✅ 已定

### 1. 电机 ✅ 已定

**决定：6374 BLDC 140KV × 3**

- 低转速高扭矩，专职提供旋转
- 不再承担发射动力功能
- KV 从 170-190 降至 140

### 2. 传动方式 ✅ 已定

**决定：6374 外转子壳面直驱**

- 电机外转子壳面（Φ63mm）嵌入筒壁
- 壳面与筒内壁齐平，直接接触球面
- 无摩擦轮、无联轴器

### 3. 布局 ✅ 已定

**决定：120° 均布嵌入筒壁，三电机同速正旋**

- 电机嵌入发射筒结构
- 筒内径 ~220mm（5 号球通过）
- 筒外径 ~290mm（含电机嵌入段）
- 筒长 ~300mm
- 三电机同转速，球绕行进轴旋转（正旋/膛线旋转）
- 每个电机在接触点的推力方向一致指向出球方向
- 旋转方向与出球发射方向一致
- 不做差速弧线，专注提供稳定可控的正旋

### 4. 支架材料 ✅ 已定

**决定：按所需结构强度确定**

- 以强度为唯一判据，不限定工艺
- 筒体需承受电机扭矩 + 球面接触力
