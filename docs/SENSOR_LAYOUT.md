# SENSOR_LAYOUT.md

## P0 最小传感器点位

### 1. 管道有球检测 `PIPE_BALL_SENSOR`
- **作用**：判断球仓 / 管道内是否还有球
- **安装位置**：多球管道下段，单球分离机构上方
- **触发条件**：检测到待供给球
- **关联故障码**：`NO_BALL_IN_PIPE`

### 2. 发射位到球检测 `BALL_IN_POSITION`
- **作用**：确认单球已到达三轮前待发位
- **安装位置**：三轮入口前导向段末端
- **触发条件**：球体进入待发位置
- **关联故障码**：`BALL_NOT_IN_POSITION`

### 3. 出球检测 `EXIT_SENSOR`
- **作用**：确认发射动作已完成
- **安装位置**：三轮后方 / 出球口
- **触发条件**：球体离开发射口
- **关联故障码**：`EXIT_TIMEOUT`

### 4. 急停输入 `ESTOP`
- **作用**：人工立即停机
- **安装位置**：操作员易触达位置
- **触发条件**：急停按钮按下
- **关联故障码**：`ESTOP_ACTIVE`

## 可选增强
- 闸门原点 / 到位检测
- 驱动温度检测
- 电源电压检测

## P0 互锁逻辑
1. `PIPE_BALL_SENSOR = ready` -> 允许进入 `FEED_READY`
2. `BALL_IN_POSITION = active` -> 允许进入 `FIRE`
3. `EXIT_SENSOR = active` -> 允许进入 `RELOAD`
4. 任一传感器超时 / 异常 -> 进入 `FAULT`
