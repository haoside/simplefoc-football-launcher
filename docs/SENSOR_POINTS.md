# 传感器点位表 v1

## 1. 管道有球检测 `PIPE_BALL_SENSOR`
- 位置：多球管道下段 / 分离机构上方
- 作用：判断是否还有球可供给
- 触发结果：无球时禁止进入供球流程

## 2. 发射位到球检测 `BALL_IN_POSITION`
- 位置：三轮入口前的待发位
- 作用：确认单球已到位
- 触发结果：仅在该信号有效时允许 `FIRE`

## 3. 出球检测 `EXIT_SENSOR`
- 位置：轮组后方 / 出球口
- 作用：确认发射动作已完成
- 触发结果：超时未触发则判定异常，进入 `FAULT`

## 4. 急停 `ESTOP`
- 位置：操作侧易触达位置
- 作用：人工立即停机
- 触发结果：切断驱动使能 / 动力回路，并进入 `ESTOP`

## 5. 温度检测（建议）
- 位置：驱动板、电机壳体
- 作用：过热保护
- 触发结果：超阈值降级或停机

## 6. P0 最小互锁逻辑
1. `PIPE_BALL_SENSOR = ready` -> 允许进入 `FEED_READY`
2. `BALL_IN_POSITION = active` -> 允许进入 `FIRE`
3. `EXIT_SENSOR = active` -> 允许进入 `RELOAD`
4. 任一传感器超时 / 异常 -> 进入 `FAULT`
