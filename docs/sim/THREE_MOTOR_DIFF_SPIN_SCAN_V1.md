# THREE_MOTOR_DIFF_SPIN_SCAN_V1

## 目的
建立 A/B/C 三电机差速扫描口径，支撑 180° 范围侧旋、左弧/右弧/上旋/下旋轨迹族验证。

---

## 坐标定义（临时）
在总装坐标未完全锁定前，先按短圆管出口视角定义：

- **A**：上方电机 / 12 点方向
- **B**：左下电机 / 8 点方向
- **C**：右下电机 / 4 点方向
- **+Z**：短圆管出口轴线 / 发射方向

后续如果 CAD 坐标重命名，只改映射，不改控制矩阵。

---

## 差速扫描变量

| 变量 | 建议初始范围 |
|---|---:|
| 基准 RPM | 2500 / 3500 / 4500 / 5500 |
| 差速 ΔRPM | 0 / 200 / 400 / 700 / 1000 |
| preload | 2 / 2.5 / 3mm |
| friction | 0.55 / 0.65 / 0.75 |
| ramp | 5 / 6 / 7s |

---

## 初始模式表

| 模式 | A | B | C | 预期 |
|---|---:|---:|---:|---|
| straight | base | base | 直线 |
| left_curve | base+Δ | base | base-Δ | 左弧 |
| right_curve | base-Δ | base | base+Δ | 右弧 |
| topspin_drop | base+Δ | base+Δ/2 | base+Δ/2 | 下坠 |
| backspin_lift | base-Δ | base+Δ/2 | base+Δ/2 | 上浮 |
| diagonal_left_drop | base+Δ | base-Δ | base | 左弧+下坠 |
| diagonal_right_drop | base | base-Δ | base+Δ | 右弧+下坠 |

---

## 需要输出的仿真结果
每组组合必须输出：

1. 出球速度
2. 自旋强度
3. 自旋轴方向
4. 横向偏移
5. 最高点
6. 落点距离
7. 打滑风险
8. 电流风险
9. 是否进入不可控区间

---

## 打滑阈值初判
若同时满足以下条件，先标为高风险：

- ΔRPM ≥ 700
- preload < 2.5mm
- friction ≤ 0.55

若出现：

- 横向偏移不随 ΔRPM 单调变化
- 出球速度突然下降
- 电流峰值异常升高

则标记为不可控区间。
