# SHORT_TUBE_V1_2A_ENGINEERING_REVIEW

## 输入
接设计师 v1.2a：

- `cad/assembly/p0-single-ball-launcher-v3/outrunner_embedded_short_tube_v1_2a.scad`
- `docs/mechanical/OUTRUNNER_EMBEDDED_SHORT_TUBE_V1_2A.md`

v1.2a 新主线基准：
- 中心为 **76mm 短圆管 / 短环**，不是长筒
- 3 个 6354 外转子环抱短圆管
- 电机嵌入式 / 夹持式安装
- 发射方向 = **短圆管出口轴线 +Z**
- 保留 `inner_d=232`、压紧 `2–3mm`、`45×45` 单端固定

---

## 1. 短圆管 `76mm` 是否够导向

### 当前判断
`76mm` 不能按“长导向筒”理解，只能作为：
- 出口轴线定义件
- 球通过中心通道的短约束
- 三电机接触区的结构基准

### 工程风险
- 入口姿态依赖前级供球/导入机构
- 出口姿态主要由三电机同步性和接触稳定性决定
- 不能指望 76mm 管长单独修正球姿态

### 结论
**可作为主线短环基准通过**，但必须补前级导入/姿态约束，不把 76mm 当长导向能力。

---

## 2. 三电机嵌入口袋是否互相干涉

### 参数复核
| 项 | 当前值 |
|---|---:|
| 电机直径 | 63mm |
| 电机长度 | 54mm |
| 口袋切向宽度 | 72mm |
| 口袋轴向长度 | 81mm |
| 口袋径向深度 | 92mm |
| 三工位角度 | 120° |

### 当前判断
从 v1.2a 参数看，单个口袋包络够用；三工位 120° 对称下，互相硬干涉风险低于旧长筒外扩方案。

### 待实测/仿真确认
- 真实电机端盖凸台
- 线束出口方向
- 螺钉头高度
- 夹持座外轮廓

### 结论
**口袋互相干涉暂通过**，真实风险转移到端盖/线束/夹持座细节。

---

## 3. 外转子旋转包络与夹持座间隙

### 当前口径
- `spin_clearance = 3mm`
- `rubber_sleeve_t = 1.5mm`
- 默认压紧 `2.5mm`

### 当前判断
`3mm` 是最低安全线，不是舒适余量。考虑：
- 包胶厚度误差
- 转子动平衡偏摆
- 打印/加工误差
- 装配偏心

当前仍有卡边风险。

### 结论
**不阻塞主线，但列为 Phase 2 第一风险。**
后续必须扫描：
- 包胶 `0 / 1 / 1.5 / 2mm`
- 压紧 `2 / 2.5 / 3mm`
- 偏摆余量 `0.5 / 1 / 1.5mm`

---

## 4. `45×45` 单端固定是否能接主框架

### 当前判断
v1.2a 保留 owner 指定的 `45×45`，并且主安装位置偏向短管后侧，符合“单端固定”口径。

### 风险
- 主框架连接面还没锁
- 工具进入空间未验证
- 单端固定对抗三电机反力矩时，局部刚度要复核

### 结论
**接口通过，刚度待主框架侧验证。**

---

## 5. 出口轴线是否符合总装发射方向

### 当前判断
v1.2a 明确：
- `+Z` 为短圆管出口方向
- 发射方向以出口轴线为准
- 三电机只负责加速/旋转，不重新定义发射基准

这与 owner 最新纠偏一致。

### 结论
**通过。后续 3D 仿真与总装必须统一用短圆管出口轴线。**

---

## 总结
v1.2a 可以作为新的主线基准，替换 v1.2 长筒理解。

### 接受项
- 76mm 短圆管 / 短环主线
- 三电机嵌入式环抱
- 45×45 单端固定
- `+Z` 出口轴线为发射方向
- 50m / 180° 控制继续走仿真

### 当前不通过制造
仍需继续：
1. 运动包络扫描
2. 包胶/压紧联动
3. 线束/端盖空间复核
4. 单端固定刚度复核
5. 50m 初速/RPM 反推
