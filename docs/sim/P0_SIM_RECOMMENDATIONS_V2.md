# P0 Simulation Recommendations v2

## Purpose

Use lightweight numerical simulation to decide the next engineering direction for the 3-wheel SimpleFOC football launcher.

The current P0 model is not final physics. It is a screening tool for parameter tradeoffs before real field calibration.

## Key finding

The initial `3000 rpm` software clamp is safe for bring-up but too conservative for practical football training passes.

With nominal assumptions:

- ball: 0.43 kg, 220 mm
- wheel: 110 mm
- friction coefficient: 0.65
- preload: 10 mm
- ramp: 5 s

The first model gives:

| preset | wheel RPM range | estimated launch speed | estimated range |
|---|---:|---:|---:|
| straight pass | 2604 / 2604 / 2604 | 13.2 m/s | 7.2 m |
| light curve assist | 2354–2854 | 13.2 m/s | 8.1 m |
| standard curve assist | 2415–3000 | 14.0 m/s | 9.8 m |

This is suitable for safe bench / near-field P0 validation, not yet for full training-field crossing passes.

## Recommendation

### Keep 3000 rpm for P0-A safety bring-up

Use this phase for:

1. no-ball spin-up
2. single-wheel closed loop
3. three-wheel sync
4. e-stop validation
5. low-speed ball launch

### Prepare 4000 rpm envelope for P0-B field tuning

Do not immediately run it. Reserve it in software/hardware design:

- firmware clamp: configurable `3000 / 4000 / 5000 rpm`
- driver current limit: configurable per test stage
- ramp: default 5 s, increase to 6 s for high RPM
- protective cover mandatory before ball tests

### Consider 130 mm wheel option

A larger wheel reduces required RPM for the same surface speed. Mechanical design should keep wheel diameter replaceable around:

- 110 mm: safe bring-up default
- 130 mm: likely better P0 field candidate
- 150 mm: reserve only if enclosure/torque/current allow

## P0 field-test sequence

1. `RPM_CAP=3000`, straight pass only.
2. `RPM_CAP=3000`, light curve only.
3. Confirm current peak, wheel slip, ball surface wear, protective cover integrity.
4. If stable, unlock `RPM_CAP=4000` in software only after e-stop and current logs pass.
5. Re-run straight / light curve / standard curve.

## Parameters to measure and feed back

| measurement | why it matters |
|---|---|
| actual wheel RPM under ball contact | detects slip/load drop |
| exit speed by radar/phone slow motion | calibrates contact efficiency |
| current peak per motor | validates power protection v1 |
| ball lateral offset | calibrates Magnus/spin model |
| preload setting | finds consistency vs. deformation tradeoff |
| wheel/ball temperature and wear | safety and repeatability |

## Product decision

For now, freeze:

- P0-A: `110 mm wheel / 3000 rpm cap / 5 s ramp`
- P0-B candidate: `130 mm wheel / 4000 rpm cap / 6 s ramp`

P0 procurement should avoid parts that make `4000 rpm` impossible later, even if P0-A starts at `3000 rpm`.
