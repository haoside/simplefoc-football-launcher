## Phase 1：几何干涉快速复核

### 1. 管体与球道
| 项 | 当前值 | 结论 |
|---|---:|---|
| 球直径 | 220mm | size-5 名义值 |
| 管内径 | 232mm | 球道余量 12mm |
| 管壁厚 | 8mm | 嵌入式口袋仍需验证局部强度 |
| 管长 | 190mm | 偏短，适合短导向/出口基准表达 |

判断：
- 管内径口径可继续保留。
- 管长 190mm 与“短圆管/出口基准”一致，不再按长发射筒理解。

### 2. 电机嵌入口袋
| 项 | 当前值 | 结论 |
|---|---:|---|
| 电机直径 | 63mm | 6354 名义值 |
| 电机长度 | 54mm | 6354 名义值 |
| 口袋径向深度 | `tube_wall + motor_d + 18 = 89mm` | 能覆盖电机外壳包络，占位偏保守 |
| 口袋切向宽度 | `motor_len + 18 = 72mm` | 给 54mm 电机长度留 18mm 余量 |
| 口袋轴向长度 | `motor_d + 20 = 83mm` | 给 63mm 直径留 20mm 余量 |
| 内侧 relief | 5mm | 有基础软化边界 |

判断：
- 从 CAD 参数看，口袋包络目前足够，不是第一阻塞。
- 真实阻塞会来自：端部固定件、线束/端盖、护罩、工具空间，而不是电机外壳本体。

### 3. 压紧与旋转安全间隙
| 项 | 当前值 | 结论 |
|---|---:|---|
| 默认压紧 | 2.5mm | 落在 2–3mm 目标中位 |
| 包胶厚度 | 1.5mm | 高风险，需要和压紧量联动校核 |
| spin_clearance | 3mm | 刚好等于最低安全线 |

判断：
- `spin_clearance = 3mm` 是最低线，后续如果包胶偏厚、动平衡偏摆、装配误差叠加，可能不够。
- 当前不能直接进制造，必须先做运动包络。

### 4. 45×45 单端安装接口
| 项 | 当前值 | 结论 |
|---|---:|---|
| 电机端孔距 | 45×45 | owner 指定，先作为模板 |
| 主安装孔距 | 45×45 | 可并线 |
| 孔径 | 5.5mm | M5 间隙孔口径 |
| 主安装垫尺寸 | 86×86×10 | 有基础边距 |

判断：
- 45×45 单端安装接口可以作为主线接口进入 v1.2。
- 待实测点：真实 6354 端面孔距 / 凸台 / 螺纹深度。

## 几何干涉结论
当前 v1.2 可以进入主线并线，但只作为：

**嵌入式圆管 + 3 外转子 + 45×45 单端安装 的仿真验证版。**

不可直接转制造。

### 暂时通过
- 管内径 `232`
- 电机嵌入口袋包络
- 45×45 单端安装口径
- 圆管出口为发射方向基准
- 50m / 180° 控制拆给仿真验证

### 当前风险
1. `spin_clearance = 3mm` 余量偏紧
2. `rubber_sleeve_t = 1.5mm` 可能把有效压紧推高
3. 电机翻转 180° 后线束/端盖空间未实测
4. 防护罩、螺钉头、工具进入空间未闭环
5. 50m 目标需要 RPM / 初速反推，不能靠结构外形判断

- 不回退到外侧外挂、不恢复粉色独立环
