# 环境仿真结果 v1

> 说明：这是工程近似仿真，用于比较环境/风向对轨迹的影响，不是 CFD 高精度结果。

## 仿真输入
### 环境
- `E0`：20°C / 50% / 0m/s
- `E1`：16°C / 60% / 2m/s
- `E1_leftwind`：16°C / 60% / 左侧风 2m/s
- `E1_rightwind`：16°C / 60% / 右侧风 2m/s
- `E2`：6°C / 45% / 4m/s
- `E2_leftwind`：6°C / 45% / 左侧风 4m/s

### 预设
- `straight_pass`：15.0 m/s，10°
- `light_left_curve`：15.0 m/s，12°
- `standard_left_curve`：16.5 m/s，14°
- `standard_right_curve`：16.5 m/s，14°

## 工程解读
1. **温湿度本身影响较小**：E0 / E1 / E2 在无侧风时，射程和落地速度变化不大。
2. **侧风影响明显大于温湿度**：2m/s 左侧风会把标准左弧线横向偏移从 0.49m 放大到 0.64m；4m/s 左侧风会放大到 0.80m。
3. **导向筒更应服务于“初始一致性”**，不是抵抗侧风；侧风下仍需要靠 preset 和落点修正。
4. **标准弧线更适合做 P0 主验证模式**：相对轻弧线，横向偏移更明显，更容易看出环境扰动。
5. **P0 优先验证顺序**：E0 -> E1 -> E1 侧风 -> E2。

## 当前建议
- 现场基线先用：`E0 + straight_pass`、`E0 + standard_left_curve`
- 常规训练验证：`E1 + standard_left_curve/right_curve`
- 抗扰动验证：`E2_leftwind + standard_left_curve`
