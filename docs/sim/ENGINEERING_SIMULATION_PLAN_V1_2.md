# ENGINEERING_SIMULATION_PLAN_V1_2

## 目标
把 v1.2 的静态结构，逐步推进到可验证的 3D / 动力学仿真包。

结构输入：
- `outrunner_embedded_launch_tube_v1_2.scad`

性能目标：
- 发射距离：50m
- 发射弧线：180° 可控
- 三外转子嵌入发射圆管
- 6354 外转子直接接触球

---

## 仿真策略：由简入繁，多轮更新

### Phase 0：参数冻结
必须先确认这些参数：

| 参数 | 当前值 | 状态 |
|---|---:|---|
| 球直径 | 220mm | 暂定 |
| 发射管内径 | 232mm | 暂定 |
| 电机外径 | 63mm | 待实测 |
| 电机长度 | 54mm | 待实测 |
| 压紧量 | 2–3mm | 待验证 |
| 包胶厚度 | 1.5mm | 待验证 |
| 主安装孔距 | 45×45 | owner 指定 |

输出：
- 参数表
- 实测缺口列表

---

### Phase 1：几何干涉仿真
目标：先确认装得下。

检查项：
1. 三个 6354 电机与发射圆管是否干涉
2. 外转子嵌入窗口是否覆盖接触带
3. 电机翻转 180° 后线束/端面是否有空间
4. 45×45 单端安装板是否与外框干涉
5. 防护罩孔位、螺钉头、工具进入空间是否足够

通过标准：
- 旋转包络无硬干涉
- 静态间隙 ≥ 3mm
- 工具进入空间 ≥ 8mm
- 45×45 孔位可接主框架

失败路径：
- 改窗口尺寸
- 改嵌入深度
- 改防护罩孔位
- 改安装板避让形状

---

### Phase 2：运动包络仿真
目标：确认转得起来，球过得去。

检查项：
1. 外转子 + 包胶最大旋转包络
2. 球体通过时与三外转子的接触/挤压关系
3. 2/2.5/3mm 三档压紧量下的通道变化
4. 防护罩与旋转外壳距离

通过标准：
- 包胶后旋转外径仍不碰固定结构
- 最小旋转安全间隙 ≥ 3mm
- 球不会被硬卡死
- 2–3mm 压紧区间可通过调节实现

失败路径：
- 减小包胶厚度
- 调整 motor_center_r
- 加大窗口/口袋
- 增加微调槽

---

### Phase 3：50m 初速反推
目标：先算理论需要多快。

输入：
- 球质量
- 出射角
- 空气阻力模型
- 目标距离 50m

输出：
- 所需初速度范围
- 电机表面线速度范围
- 推荐 RPM 区间

通过标准：
- 推荐 RPM 在 6354 可承受范围内
- 表面线速度不超过包胶安全限制
- 初速目标与台架测速可验证

失败路径：
- 增加电机转速
- 增加接触长度
- 改包胶材料
- 调整发射角
- 降低距离目标或分级目标

---

### Phase 4：接触摩擦仿真
目标：确认电机能把能量传给球。

变量扫描：
- 压紧量：2 / 2.5 / 3mm
- 包胶厚度：0 / 1 / 1.5 / 2mm
- 摩擦系数：低 / 中 / 高
- 三电机同步误差：0 / 3 / 5 / 10%

输出：
- 打滑风险
- 球损伤风险
- 电机负载估算
- 推荐压紧量与包胶厚度

通过标准：
- 打滑可控
- 球面损伤风险可接受
- 电机负载不过载
- 动平衡风险可控

失败路径：
- 改包胶材料
- 调整压紧量
- 改接触带宽度
- 增加测速闭环

---

### Phase 5：180° 弧线控制仿真
目标：验证可控轨迹族，而不是只看直线发射。

控制变量：
- 三电机独立转速
- 左右差速
- 上下差速
- 整体发射角
- 球自旋方向

输出：
- 轨迹族图
- 转速组合表
- 左弧/右弧/上旋/下旋控制边界
- 不稳定区间

通过标准：
- 能形成稳定可重复的轨迹变化
- 不出现失控打滑
- 不超过电机/包胶安全边界

失败路径：
- 加出球测速
- 加 IMU/视觉反馈
- 调整三电机角度
- 调整控制算法
