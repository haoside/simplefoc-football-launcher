# FOOTBALL_MATCH_SCENARIO_MATRIX_V1

## 目标
把 v1.2a 的仿真主线从“机械能发射”切到“足球比赛实战可用轨迹”。

核心目标：
- 3 电机用于三点差速侧旋，不只是加速
- 先验证 180° 范围可控弧线轨迹族
- 再追 50m 距离增强

---

## 实战场景矩阵

| 场景 | 用途 | 目标轨迹 | 关键控制量 | 初始优先级 |
|---|---|---|---|---|
| 直线强射 | 门将/防守反击长传 | 低偏移、高速度 | A=B=C 高速 | P0 |
| 内脚背左弧 | 绕过人墙/传中 | 左弧、可控落点 | A 高 / B 中 / C 低 | P0 |
| 内脚背右弧 | 绕过人墙/传中 | 右弧、可控落点 | A 低 / B 中 / C 高 | P0 |
| 下坠球 | 越过障碍后快速下落 | 上旋/下压 | 上方接触带更高线速度 | P1 |
| 挑高球 | 过顶/吊射 | 下旋/上浮 | 下方接触带更高线速度 | P1 |
| 斜向弧线 | 边路传中/斜传 | 横向 + 垂向复合旋转 | 三电机非对称差速 | P1 |
| 低平快弧线 | 地面附近快速弯曲 | 低角度、强侧旋 | 高线速度 + 中等差速 | P2 |
| 50m 长传 | 实战长距离转移 | 足够初速 + 可控偏移 | 高 RPM + 稳定摩擦 | P2 |

## 第一阶段测试顺序

### Stage A：直线基准
目的：先确认三点接触一致。

- A/B/C 同速
- 记录出球速度、射程、偏移
- 若直线都偏，先查三点压紧/包胶/接触带，不进入侧旋

### Stage B：左右弧线
目的：确认三电机差速能稳定地产生侧旋。

- 左弧：A 高 / B 中 / C 低
- 右弧：A 低 / B 中 / C 高
- 输出横向偏移、落点散布、打滑阈值

### Stage C：上旋/下旋
目的：确认球路下坠/上浮控制能力。

- 扫描上方/下方接触带差速
- 记录飞行高度、下落点、是否擦碰护罩

### Stage D：复合弧线
目的：接近比赛中的传中、绕人墙和斜线球。

- 三电机非对称差速
- 输出轨迹族边界
- 记录不可控区间

---

## 工程约束
为了让实战轨迹可重复，结构必须优先保证：

1. A/B/C 三接触带几何一致
2. A/B/C 三点压紧量可独立校准
3. 每个电机独立测速、独立调速、独立散热
4. 防护罩不干扰侧旋后的出球偏转
5. 圆管出口轴线仍是发射基准，不被差速逻辑改写

---

## 当前结论
v1.2a 后续仿真优先级调整为：

**实战轨迹族 > 三点差速控制 > 50m 距离增强 > 制造细节。**

50m 是距离目标，不再压过侧旋可控性。
