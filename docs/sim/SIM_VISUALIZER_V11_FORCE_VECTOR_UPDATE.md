## 新增内容

### 1. A/B/C 合力矢量分解图
新增 2D SVG 分解图：

- A 上方电机力矢量
- B 左下电机力矢量
- C 右下电机力矢量
- 合力方向箭头
- ΔRPM 与 side-spin 指标

用途：
- 评审三电机共同作用方向
- 观察 A/C 差速导致的左右偏转
- 判断 B 电机对稳定/复合侧旋的参与

### 2. 3D 合力矢量箭头
3D 场景新增：

- 三个电机指向球心的局部分力箭头
- 从短发射管出口附近出发的橙色合力箭头
- 合力箭头跟随 A/B/C RPM 实时变化

### 3. 嵌入发射管视觉强化
3D 发射管改为半透明，三电机颜色加重，突出：

- 电机不是外挂
- 电机嵌入短发射管周向
- 短发射管是结构基准，不是保护罩

## 当前合力模型
当前仍是工程筛选级简化模型：

```text
force_heading = manual_trim + lateral(A-C) + stabilizer(B)
```

其中：

- A/C 差速给左右方向主分量
- B 相对 A/C 均值给稳定/复合修正
- 手动微调代表总装角度修正

后续实测后再把该模型替换为标定曲线。
