# SIM_VISUALIZER_V9_UPDATE

## 目标
把交互式仿真从“统一转速 + 单条轨迹”升级为足球比赛实战口径：

- v1.2a：76mm 短圆管 / 短环
- 发射方向：短圆管 `+Z` 出口轴线
- 三电机 A/B/C 独立差速
- 180° 范围侧旋轨迹族
- 50m 作为距离增强目标，而不是唯一主目标

---

## Widget 更新
Widget：`仿真验证交互可视化`

版本：v9

文件：
- `widgets/sim-visualizer.html`

---

## 新增交互

### 1. 足球实战预设
新增场景：

- straight 强射基准
- 左弧 / 绕人墙
- 右弧 / 传中
- 下坠球
- 挑高球
- 斜向复合弧线
- 50m 长传增强

### 2. A/B/C 三电机独立 RPM 修正
新增：

- A 上方电机修正
- B 左下电机修正
- C 右下电机修正

A/B/C 临时坐标：
- A：上方 / 12 点
- B：左下 / 8 点
- C：右下 / 4 点

### 3. preload 回归机械口径
preload 控制范围改为：

```text
2.0mm ~ 3.0mm
```

不再沿用旧仿真里的 5~15mm 宽泛区间。

---

## 新增可视化

### 1. 轨迹族同屏
2D 图中同时显示：

- 直线
- 左弧
- 右弧
- 下坠
- 挑高

主轨迹高亮，轨迹族淡显。

### 2. 3D 轨迹族
3D 场景中同步显示多条实战轨迹族，便于评审：

- 180° 扇区内落点趋势
- 横向偏移方向
- 防护罩避让方向
- 50m 目标外圈关系

### 3. A/B/C RPM 分配
RPM 面板改为：

- A 上方
- B 左下
- C 右下

并显示：

- A/B/C 当前 RPM
- ΔRPM
- side-spin rps
- peak current estimate

---

## 风险判断更新
新增不可控区间初判：

```text
ΔRPM >= 700
preload < 2.5mm
friction <= 0.55
```

同时满足时标记为高风险：

- 打滑风险
- 侧旋不可控风险
- 实测前需收缩窗口

---

## 结论
v9 之后，仿真主线正式切到：

**足球实战轨迹族 > 三点差速侧旋 > 50m 距离增强。**

4. 把实测回填与 A/B/C 三电机校准关联起来
