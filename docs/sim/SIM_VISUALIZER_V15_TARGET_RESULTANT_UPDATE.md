## 主要改动

### 1. 合力方向锁定目标方向
原 v14/v13 里合力方向会受 A/C 差速影响发生水平偏转。  
现在修正为：

```text
force_heading = target_heading
```

即：
- `目标方向` 控制初始瞄准方位
- A/B/C 差速只进入侧旋模型
- 不再用差速改写初始发射方向

### 2. 侧旋与瞄准解耦
新口径：

| 项 | 作用 |
|---|---|
| 目标方向 | 决定初始合力瞄准方向 |
| 三电机倾斜角 | 决定向上发射角 |
| A/B/C 差速 | 决定足球飞行中的侧旋 / 弧线 |
| friction / preload | 决定能量传递和打滑风险 |

### 3. 合力图文案修正
合力矢量分解图中：

- A/B/C 不再显示为“改变合力瞄准”
- 改为“侧旋加速 / 侧旋释放”
- 橙色合力箭头表示直接指向目标方向
