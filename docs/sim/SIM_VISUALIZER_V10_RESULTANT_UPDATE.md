## 主要改动

### 1. 发射方向逻辑改为合力方向
新增合力方向估算：

```text
force_heading = manual_trim + f(A rpm, B rpm, C rpm)
```

当前简化规则：

- A/C 差速给左右方向分量
- B 参与稳定和复合侧旋修正
- `合力方向微调` 只作为总装调试偏置

### 2. 3D 箭头跟随合力方向
原来箭头固定按短管出口方向显示。  
现在随 A/B/C RPM 分配变化：

- 左弧：合力方向偏左
- 右弧：合力方向偏右
- 复合弧线：合力方向 + 侧旋轨迹共同显示

### 3. 去除保护罩口径
文案和提示已从“防护罩避让”改为：

- 无保护罩
- 短发射管出口后开放避让
- 重点看三电机嵌入发射管和出球合力方向

### 4. CAD 可视基准新增 v1.2b
新增：

- `cad/assembly/p0-single-ball-launcher-v3/outrunner_embedded_short_tube_v1_2b.scad`

v1.2b 基于 v1.2a，但：

- `show_shield_tabs = false`
- 注释改为三电机合力方向出球
- 圆管轴线保留为结构基准，不再作为唯一出球方向
