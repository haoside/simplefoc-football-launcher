# P0-B 高电流驱动候选对比 v1

## 背景

P0-A 使用 `B-G431B-ESC1` 做低功率 SimpleFOC bring-up，但它不能直接作为 6374 实球发射的最终高功率驱动。

P0-B 需要支持：

- 24V 系统
- 单路 6374 BLDC
- Hall 反馈
- SimpleFOC / 3PWM 控制
- 单路 40A 级连续能力或明确限流策略

## 候选方向

| 方向 | 优点 | 风险 | 适合阶段 |
|---|---|---|---|
| DRV8353 + 外置 MOS 自研/半自研板 | SimpleFOC 友好，电流能力可扩 | 硬件开发风险高，需要 PCB/功率设计 | P0-B / P1 |
| DRV8302 大电流开源板 | 资料多，SimpleFOC 案例多 | 市面板质量差异大，电流能力需实测 | P0-B |
| ODrive / VESC 类控制器 | 功率成熟，保护相对完整 | SimpleFOC 主线会偏离，成本更高 | 备选验证 |
| 商用伺服/BLDC 驱动器 | 可靠性高 | 成本高、接口封闭、参数调试复杂 | 后续工程化 |

## 推荐策略

1. **P0-A**：B-G431B-ESC1 只做闭环算法和通信链路。
2. **P0-B**：优先找 DRV8353/DRV8302 类 SimpleFOC 兼容高电流板。
3. **兜底**：如果高电流 SimpleFOC 板不稳定，临时用 VESC/ODrive 做功率验证，但产品主线仍保留 SimpleFOC 架构。

## 选型检查项

| 项 | 要求 |
|---|---|
| 电压 | 24V 系统，至少 30V 余量 |
| 连续电流 | 单路 ≥40A，或可稳定限流到 P0 目标 |
| 峰值电流 | 单路 ≥60A 短时 |
| 反馈 | Hall 必须支持，编码器可选 |
| 接口 | 3PWM / 6PWM / EN / FAULT / current sense |
| 散热 | 必须有散热器或风冷安装位 |
| 保护 | 过流、过温、欠压、fault 输出 |
| 软件 | SimpleFOC 示例或可快速适配 |

## 暂定结论

- 不把 B-G431B-ESC1 放进 P0-B 实球功率链路。
- P0-B 采购前必须先完成：P0-A 电流实测 + 电源保护 v1 复核。
