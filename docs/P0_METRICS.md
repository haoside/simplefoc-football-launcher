# P0 可验证指标

## 1. 单轮闭环
- 目标 RPM 范围：`500 ~ 3000 rpm`
- 稳态误差：
  - 空载：`±3%`
  - 轻接触负载：`±8%`
- 升速时间：`0 -> 2000 rpm < 1.5s`

## 2. 三轮联动
- 三轮达到 90% 目标转速同步差：`< 150 ms`
- 三轮同目标时轮间稳态偏差：`< 5%`

## 3. 射球验证
- P0 球速目标：参照专业足球运动员边路 `15 ~ 20 m` 长传需求，先验收 `15 ~ 20 m/s`
- 连续 20 次单发，成功出球：`>= 18/20`
- 连续 20 次中卡球 / 明显打滑：`<= 2 次`

## 4. 旋转控制验证
- 直射：三轮同 RPM
- 上/下旋：按预设 `ΔRPM` 组合
- 侧旋：按左右分量差速

## 5. 记录项
- `targetRpmA/B/C`
- `actualRpmA/B/C`
- `busVoltageV`
- `currentA`
- `faultCode`
- `shotResult`
- `latencyMs`
- `spinMode`

## 6. 验证方式
- 足球表面做高对比标记
- 用 `240fps` 慢动作拍摄
- 记录：出球方向 / 球速 / 旋转方向 / 是否打滑
