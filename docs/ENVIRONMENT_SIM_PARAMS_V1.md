# 足球场季节环境仿真参数 v1

## 目标
为单球三轮发射器定义可重复的环境边界，用于比较不同季节环境对：
- 出球速度
- 弧线轨迹
- 横向偏移
- 落点误差
的影响。

## 季节参数集
### 春季
- 温度：12 ~ 20°C
- 湿度：45% ~ 70%
- 风速：1 ~ 4 m/s
- 基线：16°C / 60% / 2 m/s

### 夏季
- 温度：25 ~ 35°C
- 湿度：50% ~ 85%
- 风速：1 ~ 5 m/s
- 基线：30°C / 70% / 2 m/s

### 秋季
- 温度：10 ~ 22°C
- 湿度：40% ~ 70%
- 风速：1 ~ 5 m/s
- 基线：18°C / 55% / 3 m/s

### 冬季
- 温度：0 ~ 12°C
- 湿度：30% ~ 65%
- 风速：2 ~ 7 m/s
- 基线：6°C / 45% / 4 m/s

## P0 优先环境集
- `E0`：20°C / 50% / 0 m/s
- `E1`：16°C / 60% / 2 m/s
- `E2`：6°C / 45% / 4 m/s

## 风向集
- `W0`：无风
- `W1`：顺风
- `W2`：逆风
- `W3`：左侧风
- `W4`：右侧风

## 说明
- P0 先不单独拉气压变量
- 先在无风与常规训练场环境标定，再比较侧风/低温场景
