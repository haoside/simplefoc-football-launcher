# P0-A 单球台架 Bring-up 流程 v1

## 目标

在不追求高功率的前提下，验证：

1. 单电机 SimpleFOC 速度闭环。
2. 三电机同步预转。
3. 急停 / 故障保护。
4. 单球低速稳定射出。
5. 采集第一批实测数据回填仿真模型。

## 阶段 0：安全准备

- 防护罩安装完成。
- 急停按钮可触达。
- 发射方向无人。
- 台架电源限流打开。
- 不装球。

## 阶段 1：单电机空载

| 步骤 | 目标 | 验收 |
|---|---|---|
| M1-500 | wheel1 500rpm | 稳态误差 < ±5% |
| M1-1000 | wheel1 1000rpm | 无异常振动/发热 |
| M1-2000 | wheel1 2000rpm | 电流可控 |
| M1-3000 | wheel1 3000rpm | 仅短时验证 |

wheel2 / wheel3 重复同样流程。

## 阶段 2：三轮同步空载

| 步骤 | 目标 | 验收 |
|---|---|---|
| S-1000 | 三轮 1000rpm | 90% 目标转速时间差 < 150ms |
| S-2000 | 三轮 2000rpm | 无 fault |
| S-3000 | 三轮 3000rpm | 短时，记录峰值电流 |

## 阶段 3：急停 / 故障

- 运行中拍急停：驱动 EN 立即切断。
- 通信断开：进入 FAULT。
- 任一 node fault：Host 停止发射。
- 无球检测：禁止 FIRE。

## 阶段 4：低速实球

| 档位 | RPM | 目标 |
|---|---:|---|
| B1 | 1000/1000/1000 | 观察夹持/打滑 |
| B2 | 1500/1500/1500 | 低速出球 |
| B3 | 2000/2000/2000 | 记录出球速度 |
| B4 | 2600/2600/2600 | 对齐仿真 straight_pass |

## 阶段 5：轻弧线

仅在直传稳定后执行：

- `wheel1=2600`
- `wheel2=2850`
- `wheel3=2350`

记录横向偏移、球面磨损、电流峰值。

## 必记录数据

| 数据 | 用途 |
|---|---|
| wheel1/2/3 actualRpm | 检查负载掉速 |
| 单路电流峰值 | 修正电源保护 |
| 出球速度 | 回填接触效率 |
| 横向偏移 | 回填 Magnus 系数 |
| 预压缩量 | 找稳定区间 |
| 球面磨损/温升 | 安全评估 |
