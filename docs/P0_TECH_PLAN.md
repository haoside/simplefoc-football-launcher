# P0 技术任务拆分

## 目标
P0 先完成三路电机稳定闭环、三轮同步预转、单次低速稳定射出标准 5 号足球，并具备基础安全保护。

## P0-A：单电机验证
1. MCU 跑通 SimpleFOC
2. 单路三相驱动输出正常
3. Hall 反馈正常
4. 单电机速度闭环稳定
5. 电流 / 温升 / 急停验证

**通过标准**
- 可稳定启动、升速、降速、停机
- 目标 RPM 与实际 RPM 偏差可观测
- 急停后可安全停机

## P0-B：三电机同步验证
1. 三路电机独立配置
2. 三路 RPM 独立闭环
3. 三轮同步预转
4. 加减速曲线一致性验证
5. 故障联锁验证

**通过标准**
- 三轮可独立设定 target RPM
- 三轮预转状态可稳定维持
- 任一路故障可触发全局停机

## P0-C：低速射球验证
1. 入球检测有效
2. 发射状态机可跑通：待机 -> 预转 -> 就绪 -> 发射 -> 停机/故障
3. 低速夹球测试
4. 单次低速射球
5. 射球结果记录

**通过标准**
- 单次稳定射出标准 5 号足球
- 无明显卡球 / 连续误触发
- 发射后可回到安全态

## 控制状态机
- `IDLE`
- `PRESPIN`
- `READY`
- `SHOOT`
- `FAULT`
- `ESTOP`

## 建议记录字段
- `motorId`
- `targetRpm`
- `actualRpm`
- `currentA`
- `busVoltageV`
- `faultCode`
- `shotResult`
- `latencyMs`
