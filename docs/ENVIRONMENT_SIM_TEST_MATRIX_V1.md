# 环境仿真测试矩阵 v1

## 目标
在不同环境条件下验证：
- 出球速度稳定性
- 直传轨迹稳定性
- 轻弧线/标准弧线可重复性
- 风对落点和横向偏移的影响

## 预设
- `P1`：straight_pass
- `P2`：light_left_curve
- `P3`：light_right_curve
- `P4`：standard_left_curve
- `P5`：standard_right_curve

## 最小测试矩阵
| Env | Wind | Preset | 目标 |
|---|---|---|---|
| E0 | W0 | P1 | 直传基线 |
| E0 | W0 | P4 | 标准左弧线基线 |
| E1 | W0 | P1 | 常规训练直传 |
| E1 | W0 | P4 | 常规训练标准左弧线 |
| E1 | W3 | P4 | 侧风下标准左弧线 |
| E1 | W4 | P5 | 侧风下标准右弧线 |
| E2 | W0 | P1 | 低温不利环境直传 |
| E2 | W0 | P4 | 低温不利环境标准左弧线 |
| E2 | W3 | P4 | 低温 + 侧风下标准左弧线 |

## 记录项
- 环境：温度 / 湿度 / 风速 / 风向
- preset
- wheel1/2/3 target/actual
- 球速
- 横向偏移
- 落点误差
- 旋转是否清晰
- 是否打滑

## 判定重点
1. 温度变化是否明显影响球速
2. 湿度变化是否增加打滑
3. 侧风是否放大/削弱设备弧线
4. 落点偏差是否仍可记录、可调参
