# 架构决策：B-G431B-ESC1 ×3

## 决策
P0 从“STM32G431 开发板 + 单独三相驱动板”切换为：
- `ESP32-S3 ×1`（上位控制）
- `B-G431B-ESC1 ×3`（每个轮子一块）

## 原因
1. 更少板卡与接线
2. 更适合 P0 快速验证
3. 更容易把三轮闭环先跑起来
4. 更利于控制仓缩小和布线收敛

## 边界
- 一块 B-G431B-ESC1 只控制一个轮子
- 共需要 3 块
- P0 先按中等限流验证，不一开始拉满功率

## 影响
### BOM
- 删除：STM32G431 开发板 ×3
- 删除：单独三相驱动板 ×3
- 新增：B-G431B-ESC1 ×3

### 机械
- 控制仓尺寸可进一步收缩
- 三块节点板建议靠近各自电机分散安装，避免热集中

### 固件
- wheel1 / wheel2 / wheel3 node 的定义不变
- 板级 pinout / HAL 接口需按 B-G431B-ESC1 实际资源重映射

## 当前建议
- P0 直接按 B-G431B-ESC1 ×3 推进
- 若后续遇到热/电流边界，再评估回切更大功率分立方案
