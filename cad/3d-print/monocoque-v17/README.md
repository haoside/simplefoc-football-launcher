# Monocoque Launcher — v17（推倒重做）

## 设计要点（owner 全部要求）

1. **一体化壳体**（monocoque）— 不是分离臂+座
2. **电机外壳（外转子 can）直接当滚轮** — 无额外滚轮件
3. **电机两端固定到加强筋** — 两端支撑（非悬臂），刚性好
4. **加强筋 90° 于电机轴** — 端部垂直支撑（"90度关系"）
5. **静止出线侧固定于加强筋** — 定子/出线侧固定，can 中段转动
6. 气压主线（电机 = 控旋 / 控球）

## 几何

```
球:           220mm（R 110）
管内径:        113mm
管外径:        121mm
管长:          114mm（电机 74 + 两端筋空间）
电机:          6374（can = 滚轮）
电机中心 R:    140.5mm（球R - 1mm压紧 + can R）
can:          中段当滚轮，接触球
壳体外径:      182mm
加强筋厚:      18mm（两端各一）
出线口:        底部加强筋开槽
```

## 结构剖析

```
   顶部加强筋 ═══════════  ← 90° 端盘，固定电机上端定子
      ║  ▓can▓  ║          ← can 外转子（中段滚轮）
      ║  ▓can▓  ║             接触球，转动给旋
   底部加强筋 ═══════════  ← 90° 端盘，固定电机下端定子 + 出线口
        球（中心）
```

- 电机两端（静止定子）固定到上下加强筋
- can（外转子）在两筋之间转动
- can 中段通过管壁槽接触球
- 出线从底部加强筋引出
