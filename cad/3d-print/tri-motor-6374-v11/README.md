# Tri-Motor 6374 3D Printed Parts — v11 (True 180° Axial Spider)

## 校正前几版设计

**v6-v10 错误**：电机轴向被我做成了"径向"（X 方向），应该**真正平行于管长**（Z 方向）

**v11 正确版**：
- 电机轴沿 Z（平行于管长）
- Cradle 是**盘状**，盘面垂直于 Z 轴
- 3 辐条 + 外圈在 XY 平面内
- Can 径向（X 方向）从 hub 朝球方向伸出

## 180° 电机关系

- 电机本体一端 → 固定在 hub（管外侧）
- Can 端 → 180° 反向 → 朝球方向（管内侧）
- 旋转时，can 圆柱面接触球 → 沿管轴方向产生力

## 几何参数

```
管:       IR 113 / OR 140 / L 150
Cradle 中心: 190mm 半径（管外 50mm）
盘厚:     8mm
Hub 长度: 82mm（沿 Z 轴，容纳 6374）
轮外径:   230mm
辐条:     3 条 120° 间隔
Can:      30mm 长径向
```

## 装配

```
       Z 轴（管长方向）
        ↑
   ┌────┼────┐
   │   ╱╲    │     ← 盘状 cradle（垂直于 Z）
   │  hub    │       电机轴沿 Z 穿过 hub
   │         │
   ╰────┼────╯
        │
   ══════════════
        中心管段
        │
        ↓
       球（220mm）
```

3 个 can 旋转 → 球被夹持 → 沿 Z 轴方向射出
- 同速：直线
- 差速：弧线/传中

## 打印

- 打印机：≥ 250×250×250mm
- 方向：管长方向竖直
- 支撑：cradle 外圈下方需要
