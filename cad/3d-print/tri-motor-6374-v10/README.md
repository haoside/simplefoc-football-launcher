# Tri-Motor 6374 3D Printed Parts — v10 (Integrated 180° Spider)

## 关键设计：180° 电机关系

**180° 关系** = 电机本体和 can 端在电机轴两端反向布置：
- 电机本体一端 → 管外（hub 法兰固定）
- Can 端 → 180° 反向 → 穿过管壁 → 接触球面

电机轴**径向**（垂直于管长），3 电机 120° 分布。

## 一体化结构

按 PM 要求**先做一体**（不考虑分体打印）：

```
     ┌──/─────\──┐
     │  │  │  │  │
     │ ╭┴───┴╮ │   ← 外圈
     │ │spoke│ │   ← 3 辐条
     │ ├─────┤ │   ← 中心 hub
     │ │电机  │ │
     │ ╰─────╯ │   ← can 朝管内（180° 反向）
     └──\─────/──┘
        ║  ║  ║       ← 桥接筋
   ════════════════════════
       中心管段（120mm）
```

## 几何参数

```
管内径:     113mm
管外径:     140mm
管长:       120mm
电机中心:   145mm（管壁外侧 5mm）
轮外径:     190mm
辐条:       3 条 120° 间隔
壁厚:       10mm
```

## 机制

3 个 can 旋转 → 球被夹持并加速 → 沿管轴方向射出
- 同速：直线球
- 差速：弧线球/传中（spin 旋转）

## 打印（一体化）

- 材料：PETG / ABS
- 打印机：≥ 250×250×250mm 打印空间
- 方向：管长方向竖直
- **支撑**：需要大量支撑（cradle 外侧下方）
- 填充：30%
