# Tri-Motor 6374 3D Printed Parts — v5 (Spoked Wheel Pattern)

参考图最新观察：橙色件是**明显的辐条轮样式**——外圈 + 3 条径向辐条 + 中心 hub。

## 关键修正（vs v4）

v4 缺少参考图最显著的特征——**辐条轮样式**。v5 显式加入：
- 每个电机位 = 一个完整辐条轮（外圈 + 3 辐条 + 中心 hub）
- 3 个辐条轮 120° 均布围绕中心管段
- 与参考图橙色件视觉特征一致

| 文件 | 用途 | 数量 |

| `spider_body_v5.stl` | **★ 一体化蜘蛛主体**（管+3 辐条轮+连接肋） | 1 |

## 辐条轮结构

```
       ╱─────╲       ╱─────╲       ╱─────╲
      │  外圈  │   │  外圈  │   │  外圈  │    ← 外圈（环状）
       ╲──┬──╱       ╲──┬──╱       ╲──┬──╱
          │辐│           │辐│           │辐│     ← 3 条辐条（120°）
          │条│           │条│           │条│
          └┬┘            └┬┘            └┬┘
       ╱───┴───╲       ╱───┴───╲       ╱───┴───╲
      │  hub  │   │  hub  │   │  hub  │    ← 中心 hub（容纳 6374）
       ╲──────╱       ╲──────╱       ╲──────╱
            │              │              │
            ╰──────────────┴──────────────╯
                        │
                  ══════════════
                  中心管段 (113mm 内径)
```

## 与 v4 区别

| | v4 | v5 |
|---|----|----|
| 电机座样式 | 实心凸台 | **辐条轮**（外圈+3 辐条+hub）|
| 参考图符合度 | 部分 | 高 |
| 重量 | 较重 | 较轻（减重）|
| 视觉特征 | 简化 | 完整辐条轮样式 |

## 打印建议

- 材料：PETG / ABS
- 层高：0.2mm
- 填充：30%
- **方向**：管长方向竖直打印，3 个辐条轮向上
- **支撑**：辐条下方需要桥接支撑
