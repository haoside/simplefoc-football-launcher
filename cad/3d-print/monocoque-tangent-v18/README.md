# Monocoque Tangent-Roller Launcher — v18（正确几何）

## 核心修正

**电机轴 = 切向**（垂直于径向，也垂直于管轴 Z）

owner 三条指令合并：
- "恰恰有 90 度关系" → 电机轴垂直于管
- "三电机旋转与圆管径向相切" → 旋转切向
- "电机轴不平行于管" → 否定 v17 的平行假设

## 为什么这次能发射（力学证明）

电机在 0° 位置 (140.5, 0, 0)，轴沿切向 Y：
```
接触点 (109, 0, 0)（can 内缘，球面 110mm，1mm 压紧）
r = 接触点 - 轴位置 = (-31.5, 0, 0)
ω = (0, ω, 0)（切向）
v = ω × r = (0, 0, 31.5ω)  ← +Z 方向 = 发射方向！
```

3 电机 120° 同向旋转 → 都推球 +Z → **球被摩擦驱动沿管射出**
差速 → 弧线球 / 上下旋

## 结构

```
        球（中心）
    can①↗   can②↘   ← 切向滚轮（轴 = 切线方向）
       ╲    │    ╱      can 滚动 → 推球 +Z
        支撑板（两端夹电机）
    ═══一体化壳体═══
```

- 每个电机 can 切向放置
- can 两端定子（出线侧）固定到支撑板
- 支撑板垂直于切向轴（90° 关系）
- 一体化壳体连接 3 组支撑

## 推进方式

**两种可选（已验证切向几何能发射）：**
1. **纯摩擦驱动**：3 切向滚轮直接驱动球（v18 几何支持）
2. **气压 + 控旋**：气压主推 + 切向滚轮控旋（更高速）
