# Monocoque Orthogonal-Axis Launcher — v19

## 关键澄清（owner 反馈）

> "恰恰有 90 度关系" 不意味着 电机轴垂直于管
> 实际为 空间相错 90 度平行但不相交

**3 个电机轴互相正交**（X / Y / Z 各一个）：
- Motor 1: X 方向（径向 0° 位置）
- Motor 2: Y 方向（切向）
- Motor 3: Z 方向（轴向）

互相 90°（mutually perpendicular），电机位置错开 120°，3 根轴在空间不相交（skew）。

## 力学效果

| 电机 | 轴向 | 接触点速度 | 球效果 |
|------|------|-----------|--------|
| 1 | X（径向）| 在轴线上 = 0 | 不打滑 |
| 2 | Y（切向）| 偏离轴线 = +Z | **发射球** |
| 3 | Z（轴向）| 在 XY 平面 | 自旋/弧线 |

只有 Motor 2 (Y 轴) 真正推球沿 +Z 发射；Motor 3 给自旋；Motor 1 闲置（可换位置）。
