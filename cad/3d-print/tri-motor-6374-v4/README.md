# Tri-Motor 6374 3D Printed Parts — v4 (Integrated Spider)

基于参考图：黄色件为**一体化蜘蛛结构**——中心管段 + 3 条弯曲辐臂 + 电机座，一件成型。

## 关键特征

1. **单件打印**：管 + 臂 + 电机座全部一体
2. **弯曲辐臂**：贝塞尔曲线生成有机造型
3. **3 个电机位**：120° 均布，每个容纳 6374 电机
4. **大圆孔**：电机外壳 can 从电机座伸出，穿过管壁接触球面
5. **减重凹槽**：臂中段开槽省料省打印时间

## 零件清单（3 件，仅打印需要）

| 文件 | 用途 | 数量 |
|------|------|------|
| `spider_body_v4.stl` | **★ 一体化蜘蛛主体**（管+臂+电机座） | 1 |

## 与 v3 分件方案对比

| | v3 分件 | v4 一体化 |
|---|--------|----------|
| 零件数 | 6+ | 3 |
| 装配 | 多螺栓 | 少螺栓 |
| 刚性 | 一般 | 优（一体） |
| FDM 打印 | 易 | 难（需大量支撑） |
| 调试 | 易换件 | 整体重打 |

## 打印建议

- 材料：PETG / ABS
- 层高：0.2mm
- 填充：30-40%
- **方向**：管长方向竖直打印，电机座朝上
- **支撑**：需大量支撑（弯曲臂下方）
- 打印机：建议 ≥ 250×250×250 mm 打印空间
