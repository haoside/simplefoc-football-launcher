# Tri-Motor 6374 3D Printed Parts — v1

基于参考图片方案，适配 6374 电机 × 3，120° 环形排布，标准5号足球发射器。

## 设计参数

| 参数 | 值 |
|------|-----|
| 球通道内径 | 225 mm（220mm 球 + 5mm 间隙） |
| 电机 | 6374 BLDC × 3 |
| 排布 | 120° 均匀分布 |
| 壁厚 | 4 mm |
| 滚轮直径 | 60 mm |
| 滚轮宽度 | 20 mm |
| 结构臂长度 | 120 mm |
| 框架宽度 | 80 mm |

## 零件清单

| 文件 | 用途 | 数量 |
|------|------|------|
| `frame_main.stl` | 主框架 — 3 个电机安装凸台 120° 均布 | 2（对开） |
| `motor_mount.stl` | 电机座 — 法兰夹持 6374 | 3 |
| `roller_wheel.stl` | 发射滚轮 — V 槽抓球 | 3 |
| `structural_arm.stl` | 连接臂 — 框架 ↔ 电机座 | 3 |
| `connector_flange.stl` | 法兰环 — 两半框架螺栓合体 | 2 |
| `motor_side_plate.stl` | 侧盖板 — 3 轴通过孔，增加刚性 | 1 |

## 与现有 v3 模型的关系

本零件集采用**独立滚轮+结构臂**方案，与 `cad/assembly/p0-single-ball-launcher-v3/` 的**外转子嵌入式一体结构**方案不同：

- **v3 方案**：电机外壳（外转子）直接作为接触面，电机嵌入管壁一体结构
- **本方案**：电机通过结构臂连接框架，独立滚轮装在电机轴上

两种方案可并行验证，按实际装配效果选择。

## 打印建议

- **材料**：PETG 或 ABS（强度 + 耐冲击）
- **层高**：0.2mm
- **填充**：40-60%（受力件）
- **壁厚**：4 层（1.6mm）
- **支撑**：框架半体和电机座需要少量支撑

## 修改参数

```bash
cd cad/3d-print/tri-motor-6374-v1
python3 params.py
```

修改 `params.py` 顶部参数后重新运行即可生成新 STL。
