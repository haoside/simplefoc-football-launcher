# Tri-Motor 6374 3D Printed Parts — v2 (Outer Rotor)

基于参考图修正，**电机外壳穿过管壁接触球面**。

## 设计概念

6374 电机的外转子（外壳/can）作为发射面，从管壁外侧穿过管壁上的大圆孔进入管内，外壳外缘接触球面。

```
        球 (220mm)
       /        \
      /          \
   ┌─/────────────\─┐
   │  ╱          ╲  │   ← 管内壁 + 大圆孔
   │ │  ◯  ◯  ◯  │ │   ← 3 个电机外壳穿入 (120° 均布)
   │  ╲          ╱  │
   └─\────────────/─┘
        ┃   ┃   ┃       ← 加厚凸台（向外延伸 20mm）
      ═══╬═══╬═══╬═══    ← 桥接筋（保持刚性）
        6374 电机 × 3
```

## 零件清单

| 文件 | 用途 | 数量 |
|------|------|------|
| `launch_tube.stl` | 主框架：圆管 + 3 大圆孔 + 加厚凸台 + 桥接筋 | 1 |
| `motor_can.stl` | 电机外壳套：3×V 槽增加抓球 | 3 |
| `side_plate.stl` | 端板：球通过孔 + 电机通过孔 | 2 |
| `stator_mount.stl` | 定子固定座（防止定子随 can 转动） | 3 |
| `bridge_rib.stl` | 独立加固筋（可选备用） | 3 |

## 参数

```
球: 220mm
管内径: 226mm（球通过间隙 6mm）
管壁厚: 27mm
凸台外径: 160mm（向外延伸 20mm）
电机中心距管中心: 146.5mm
电机外壳外缘接触球面
```

## 打印设置

- 材料：PETG / ABS
- 层高：0.2mm
- 填充：50-60%
- 支撑：launch_tube 需要较多支撑（凸台外侧）

## 与 monocoque v3 的差异

- **monocoque v3**（同目录其他版本）：单件一体成型，八边形卡座
- **本方案 v2**：圆管 + 凸台 + 筋，结构清晰，便于装配调试

两者并行验证。
