# Tri-Motor 6374 3D Printed Parts — v14 (CORRECTED: Radial + Flywheel Disc)

## 关键修正

**v6-v13 错误设计：**
- 电机轴向径向或平行于管，can 为圆柱壳
- Can 旋转产生的接触速度**无 Z 分量** → 球只绕管轴原地打转

**v14 正确设计：**
- 电机轴向**径向**（垂直于管）
- 端部装**平面飞轮盘**（盘面垂直于电机轴）
- 盘旋转产生的接触速度**有 Z 分量** → 球沿管射出

## 力学推导

电机轴沿径向（X），盘面在 YZ 平面：
- 盘面接触点 (x_c, y_c, z_c)，其中 y_c² + z_c² ≤ r_disc²
- 盘旋转 ω 在 +X 方向
- 接触点速度：v = ω × r = (0, -ω·z_c, ω·y_c)
- **Z 分量 v_z = ω·y_c**

3 个盘在 120° 分布，所有 Z 分量同向叠加：
- **净 Z 力 → 球沿管轴射出** ✓
