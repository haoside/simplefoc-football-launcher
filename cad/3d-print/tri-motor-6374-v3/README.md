# Tri-Motor 6374 3D Printed Parts — v3 (Spoked Cradle)

基于参考图细节修正，**电机轴向平行于管长**，使用**辐条轮电机座**独立件。

## 关键观察（参考图）

1. **橙色 3D 打印件 = 独立辐条轮电机座**（不是管壁凸台）
2. **电机轴向平行于管长**（不是径向）
3. **电机外壳穿过管壁大圆孔**接触球面
4. **外缘用螺栓固定到管壁**
5. **桥接筋**连接相邻辐条轮保持刚性

## 零件清单

| 文件 | 用途 | 数量 |
|------|------|------|
| `launch_tube.stl` | 发射管（简化：只有大圆孔 + 螺栓孔） | 1 |
| `motor_cradle.stl` | **★ 辐条轮电机座**（参考图橙色件） | 3 |
| `motor_can.stl` | 电机外壳套（V 槽） | 3 |
| `side_plate.stl` | 端板 | 2 |
| `stator_holder.stl` | 管内定子固定座 | 3 |
| `bridge_rib.stl` | 桥接筋（连接相邻 cradle） | 3 |

## 结构示意

```
       ┌─── motor_can (转子，穿过管壁)
       │
   ╔═══╪═══╗   ← motor_cradle（辐条轮，4 辐条 + 外缘）
   ║   │   ║      外缘 M5 螺栓固定到管壁
   ║ ┌─┴─┐ ║      中心 hub 容纳电机
   ║ │电机│ ║      电机轴向 || 管长
   ║ └─┬─┘ ║
   ╚═══╪═══╝
       │
   ────┼────  ← 管壁（大圆孔让 can 穿过）
   ╔═══╪═══╗
   ║   球   ║
   ╚════════╝
```

## 与 v2 区别

- v2：管壁带凸台（凸台是管壁一部分）
- v3：管壁只有孔，**独立的辐条轮**作为电机座（更接近参考图橙色件）

## 装配

1. 管上 3 个电机位大圆孔穿过电机外壳
2. motor_cradle 用 4×M5 螺栓固定到管壁（外缘）
3. 6374 电机用 4×M4 螺栓固定到 cradle 中心 hub
4. motor_can 套在电机外转子上
5. stator_holder 装在管内，固定电机定子不旋转
6. bridge_rib 连接相邻 cradle 外缘（保持刚性）
