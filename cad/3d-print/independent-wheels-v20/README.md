## 几何（单个轮辋）

```
轮外径:    160.5mm
轮内径:    115.5mm（包管外侧）
环宽:      22.5mm
环厚:      14mm（加强）
Hub OD:    71mm
Hub 长度:   80mm（沿 Z 轴，容纳电机）
辐条:      3 条 120° 间隔
螺栓:      4×M5（外圈 PCD 138mm）
```

## 装配（3 轮独立）

```
                    球
                  ↗  ↓  ↖
              轮② can can 轮①
                ↑  ↓  ↑
              can     can
              can     can
              can     can
              轮③══════════轮③
                  ║  管  ║
                  ╚═════╝
```

每个轮独立件：
- 用 4 颗 M5 螺栓固定到管（管上预埋螺母）
- 电机 can 通过管壁孔接触球
- 电机轴切向（v18 物理）

## 打印

- 材料：PETG / ABS / CF-PETG
- 方向：管长竖直
- 3 个独立轮打印后装配
