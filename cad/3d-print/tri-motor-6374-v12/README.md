## 设计哲学

参考图等比例放大到 220mm 球 + 6374 电机规格，**保持视觉比例**：

| 参数 | 参考图比例 | 放大后 |
|------|-----------|--------|
| 电机 OD | 25mm | **63mm** (×2.5) |
| 球径 | ~75mm | **220mm** (×2.93) |
| 轮外径 | 2.5×电机 | **157.5mm** |
| Hub OD | 1.15×电机 | **72.5mm** |
| 盘厚 | 0.25×电机 | **15.75mm** |
