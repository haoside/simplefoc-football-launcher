# Control Bay Plate v1

## Purpose
Mounting plate for ESP32-S3, three B-G431B-ESC1 nodes, and CAN/terminal area.

## Key dimensions
- Plate: 250 mm x 180 mm x 4 mm
- Corner mount holes: 4 x 5.5 mm
- Board standoff holes: mixed 3.2 mm / 4.0 mm provisional
- Material: 5052 / 6061, 4 mm

## Notes
- Hole positions are provisional for P0 packaging.
- Must be rechecked against actual B-G431B-ESC1 board outline after purchase.
