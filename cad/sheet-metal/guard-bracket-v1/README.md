# Guard Bracket v1

## Purpose
Flat bracket set for supporting the transparent PC guard cover.

## Key dimensions
- Main bracket plate: 220 mm x 40 mm x 4 mm
- Mount holes: 4 x 5.5 mm
- Guard holes: 3 x 4.5 mm
- Material: 5052 / 6061, 4 mm

## Notes
- P0 favors flat-plate manufacturability first.
- Final bend / standoff strategy may be added after frame spacing is confirmed.
