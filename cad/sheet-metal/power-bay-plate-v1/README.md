# Power Bay Plate v1

## Purpose
Mounting plate for 24V power unit / BMS accessories / fuse / contactor / DCDC area.

## Key dimensions
- Plate: 320 mm x 220 mm x 4 mm
- Corner mount holes: 4 x 6.0 mm
- Provisional internal mounting holes for power supply and accessories
- Material: 5052 / 6061, 4 mm
